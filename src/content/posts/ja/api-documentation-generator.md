## 📊 証明: Before & After

### ❌ Before (入力コード - Node.js Express)

```javascript
app.post("/users", (req, res) => {
  const { username, email } = req.body;
  if (!email) return res.status(400).send({ msg: "no email" });
  // DB 保存ロジック...
  res.status(201).send({ id: 1, username, email });
});
```

### ✅ After (結果 - Markdown 例示)

### **POST /users**

ユーザー会員登録API

**Request Body**
| フィールド名 | タイプ | 必須 | 説明 |
|---|---|---|---|
| `username` | String | O | ユーザーニックネーム |
| `email` | String | O | ユーザーメール (ユニーク) |

**Response (201 Created)**

```json
{
  "id": 1,
  "username": "jay_dev",
  "email": "jay@example.com"
}
```
