## (Updated) 🚀 해결책: "엑셀 수식 마스터" {#updated}

### (Updated) 🥉 Basic Version (기본형) {#updated}

빠르게 결과만 필요할 때 사용하세요.

> **역할:** 너는 20년 경력의 엑셀 전문가이자 데이터 분석가야.
> **요청:** VLOOKUP, INDEX MATCH 헷갈리시나요? 상황만 설명하면 엑셀 함수를 완벽하게 짜드립니다해줘.

<br>

### (Updated) 🥇 Pro Version (전문가형) {#updated}

디테일한 퀄리티가 필요할 때 사용하세요.

아래 **PROMPT** 내용을 복사해서 사용하세요.

> **역할 (Role):** 너는 20년 경력의 엑셀 전문가이자 데이터 분석가야.
>
> **상황 (Context):** 나는 엑셀 초보자이고, 특정 데이터를 처리하고 싶은데 어떤 함수를 써야 할지 모르겠어.
>
> **요청 (Task):**
>
> 1. 내가 설명하는 상황에 딱 맞는 엑셀 함수(Formula)를 작성해줘.
> 2. 해당 함수가 어떻게 작동하는지 단계별로 쉽게 설명해줘.
> 3. 예시 데이터를 포함해서 바로 복사해서 쓸 수 있게 해줘.

- **제약사항 (Constraints):**
  > - 복잡한 설명 대신 핵심만 간결하게 설명할 것.
- 엑셀 버전에 따른 호환성 이슈가 있다면 언급할 것.
- 결과물은 코드 블록 안에 넣어서 복사하기 쉽게 할 것.

## (Updated) 📊 증명: Before & After {#updated}

단순한 명령어와 최적화된 프롬프트의 차이는 명확합니다. 제공된 프롬프트를 사용했을 때 더 구체적이고 구조화된 답변을 얻을 수 있으며, 재수정(Re-prompting)에 드는 시간을 절약할 수 있습니다.

### (Updated) ❌ Before (입력) {#updated}

```text
A열에 있는 상품명이랑 같은 걸 B열에서 찾아서 C열 값을 가져오고 싶어.
```

### (Updated) ✅ After (결과) {#updated}

```text
=VLOOKUP(A2, B:C, 2, FALSE)

설명: A2 셀의 값을 B열에서 찾아, 매칭되는 행의 C열(2번째 열) 값을 가져옵니다.
```
