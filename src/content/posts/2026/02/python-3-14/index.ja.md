## 📊 証明：Before & After

### ❌ Before (入力)

```python
from typing import TypeVar, Generic

T = TypeVar('T')

class DataBox(Generic[T]):
    def __init__(self, data: T):
        self.data = data

def process_data(obj):
    if isinstance(obj, dict) and "status" in obj:
        if obj["status"] == "success":
            return obj.get("data")
    return None
```

### ✅ After (結果)

```python
from typing import TypeVar, Generic

# Python 3.14: TypeVarにデフォルト値（strなど）を指定可能
T = TypeVar('T', default=str)

class DataBox(Generic[T]):
    def __init__(self, data: T):
        self.data = data

def process_data(obj: dict | any) -> any:
    # Python 3.14: パターンマッチング v3でより簡潔に
    match obj:
        case {"status": "success", "data": data}:
            return data
        case _:
            return None
```
