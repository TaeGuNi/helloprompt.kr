Вы можете поиграть в неё прямо сейчас: [three-descent](https://mrdoob.github.io/three-descent/).

Портирование ретро-игр в веб перестало быть магией, доступной лишь избранным хакерам. Главный секрет кроется в жестком разделении ответственности: пусть скомпилированный через Emscripten модуль WASM забирает на себя всю "тяжелую" математику, физику коллизий и логику игры, а Three.js занимается исключительно красивой отрисовкой кадров. С помощью предложенного промпта вы сможете сэкономить десятки часов на мучительном чтении документации по интеграции памяти между JavaScript и C++, сразу получив рабочий каркас для смелых экспериментов.

```javascript
// ИИ мгновенно генерирует готовый мост между WASM и Three.js
function updateGameLogic() {
  // 1. Вызываем шаг игровой логики из WASM
  wasmModule._tick();

  // 2. Получаем указатель на буфер с позициями объектов
  const positionPtr = wasmModule._getEntitiesPositions();
  const entityCount = wasmModule._getEntityCount();

  // 3. Создаем типизированный массив для быстрого чтения памяти WASM
  const positions = new Float32Array(
    wasmModule.HEAPF32.buffer,
    positionPtr,
    entityCount * 3,
  );

  // 4. Обновляем инстансы в Three.js
  for (let i = 0; i < entityCount; i++) {
    const mesh = entityMeshes[i];
    mesh.position.set(
      positions[i * 3],
      positions[i * 3 + 1],
      positions[i * 3 + 2],
    );
  }
}
```
