## 📊 証明: Before & After

### ❌ Before (入力)

```javascript
function calc(d) {
  let t = 0;
  for (let i = 0; i < d.length; i++) {
    if (d[i].price > 100) {
      t += d[i].price * 0.9;
    } else {
      t += d[i].price;
    }
  }
  return t;
}
```

### ✅ After (結果)

```javascript
/**
 * 買い物かごの総額を計算します。
 * 100ウォン超過商品は10%割引を適用します。
 */
function calculateTotalPrice(items) {
  const DISCOUNT_THRESHOLD = 100;
  const DISCOUNT_RATE = 0.9;

  return items.reduce((total, item) => {
    const finalPrice =
      item.price > DISCOUNT_THRESHOLD ? item.price * DISCOUNT_RATE : item.price;
    return total + finalPrice;
  }, 0);
}
```

_変更事項: `calc` -> `calculateTotalPrice`, `d` -> `items`, マジックナンバーの定数化, `reduce` 使用._
