# 📝 Внимание: Уязвимость RCE в Windows Notepad (CVE-2026-20841)

- 🎯 **Рекомендуется для:** Инженеров по безопасности, системных администраторов, IT-директоров (CISO)
- ⏱️ **Затрачиваемое время:** 2 часа → Сокращено до 5 минут

> _"Самое безобидное приложение в вашей системе только что стало главным бэкдором. Как быстро объяснить это руководству и развернуть патч?"_

Недавно обнаруженная уязвимость нулевого дня (CVE-2026-20841) в классическом приложении «Блокнот» (Notepad) для Windows позволяет злоумышленникам удаленно выполнять произвольный код (RCE) через специально созданные текстовые файлы. Когда на кону стоит безопасность всей корпоративной сети, счет идет на минуты. Этот промпт поможет вам мгновенно сгенерировать понятный отчет для руководства и скрипт PowerShell для временного устранения угрозы до выхода официального патча от Microsoft.

## ⚡️ Краткая выжимка (TL;DR)

1. **Суть угрозы:** Уязвимость RCE (CVE-2026-20841) активируется при простом открытии вредоносного файла `.txt` в Блокноте.
2. **Риски для бизнеса:** Полная компрометация рабочих станций сотрудников без запроса дополнительных прав администратора.
3. **Решение ИИ:** Используйте приведенный ниже промпт для мгновенного создания плана реагирования и скриптов смягчения последствий (mitigation scripts).
