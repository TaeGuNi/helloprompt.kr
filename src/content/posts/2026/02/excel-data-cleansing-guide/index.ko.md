# 📝 지저분한 엑셀 데이터, 깔끔하게 전처리하기

- **🎯 추천 대상:** 매달 엑셀 취합으로 고통받는 마케터, 기획자, 실무 담당자
- **⏱️ 소요 시간:** 30분 → 1분 단축
- **🤖 추천 모델:** ChatGPT (Advanced Data Analysis 가능 모델 권장), Claude 3.5 Sonnet

- ⭐ **난이도:** ⭐⭐☆☆☆
- ⚡️ **효과성:** ⭐⭐⭐⭐⭐
- 🚀 **활용도:** ⭐⭐⭐⭐⭐

> _"주소랑 우편번호가 한 셀에 섞여 있어서, 스페이스바 누르며 하나씩 지우고 계신가요?"_

데이터 분석보다 더 오래 걸리는 게 바로 데이터 전처리(Data Cleaning)입니다. 부서마다 제각각인 날짜 형식, 띄어쓰기가 엉망인 주소, 괄호와 특수문자가 난무하는 텍스트까지.

함수를 찾고 구글링하는 시간조차 아까울 때, AI에게 엑셀 파일 규칙을 찾아달라고 하거나 정제된 수식을 단번에 얻어낼 수 있습니다. 이 프롬프트 하나면 지긋지긋한 데이터 클렌징 작업을 압도적으로 단축할 수 있습니다.

---

## ⚡️ 3줄 요약 (TL;DR)

1. 구글링으로 함수나 매크로를 찾는 대신, AI에게 데이터 패턴을 알려주고 해결책을 직접 요구하세요.
2. 빠른 채우기(Flash Fill)나 정규식, 혹은 엑셀 함수를 상황에 맞게 1초 만에 추천받을 수 있습니다.
3. 실무에 즉시 적용 가능한 수준으로 데이터 클렌징 시간을 90% 이상 절감합니다.

## 🚀 해결책: "엑셀 데이터 클리너"

### 🥉 Basic Version (기본형)

빠르게 결과만 필요할 때 사용하세요.

> **역할:** 너는 10년 차 엑셀 데이터 처리 및 함수 전문가야.
> **요청:** 엑셀에서 주소와 우편번호가 섞여 있는 데이터를 분리하고 싶어. 가장 쉬운 해결 방법을 알려줘.

<br>

### 🥇 Pro Version (전문가형)

디테일한 퀄리티와 정확한 함수/단축키 가이드가 필요할 때 사용하세요.

> **역할 (Role):** 너는 대기업 재무팀에서 근무하는 10년 차 엑셀 데이터 처리 및 매크로 전문가야.
>
> **상황 (Context):**
>
> - 배경: 수백 명의 고객 설문조사 데이터를 취합했는데, A열(주소 데이터)의 형식이 엉망진창이야.
> - 목표: A열의 데이터 예시 `[서울시 강남구 테헤란로 123 (06234)]`를 분석해서, B열에는 `주소(서울시 강남구 테헤란로 123)`를 남기고, C열에는 `우편번호(06234)`만 남도록 완전히 분리해야 해.
>
> **요청 (Task):**
>
> 1. 위 상황을 해결할 수 있는 가장 효율적이고 간단한 방법(엑셀 내장 기능 또는 함수)을 1순위로 제시해 줘.
> 2. 만약 함수(LEFT, RIGHT, MID, FIND 등)를 사용해야 한다면, 초보자도 그대로 복사해서 붙여넣을 수 있는 정확한 수식을 제공해 줘.
> 3. 빠른 채우기(Ctrl+E, Flash Fill)로 해결 가능한 문제라면, 작업 순서를 1, 2, 3단계로 명확히 나누어 설명해 줘.
>
> **제약사항 (Constraints):**
>
> - 설명은 초보자가 읽고 바로 따라 할 수 있도록 친절하고 직관적이어야 해.
> - 불필요한 서론은 빼고 바로 해결 방법부터 마크다운 리스트 형태로 제시해 줘.
>
> **주의사항 (Warning):**
>
> - 최신 엑셀 버전(Microsoft 365)을 기준으로 하되, 구버전 엑셀에서는 안 되는 기능이라면 반드시 예외 상황과 대체 함수를 언급해 줘.
> - 괄호가 없는 데이터가 섞여 있을 경우 발생하는 오류(예: #VALUE!)를 방지할 수 있는 예외 처리(IFERROR 등)도 수식에 포함해 줘.

## 💡 작성자 코멘트 (Insight)

이 프롬프트의 핵심은 단순한 함수 질문을 넘어, "내가 가진 데이터의 예시 패턴"을 정확히 던져주는 것입니다.
실무에서 엑셀 작업을 할 때 가장 흔히 겪는 문제가 바로 '예외 케이스(어떤 셀은 괄호가 없고, 어떤 셀은 띄어쓰기가 두 번 되어 있는 등)'입니다.

AI에게 질문할 때 "이런 예외도 있을 수 있다"라고 넌지시 던져주면, `IFERROR`나 복잡한 배열 수식까지 고려하여 한 번에 완벽하게 작동하는 코드를 짜줍니다. 특히 **'빠른 채우기(Ctrl+E)'** 같은 엑셀의 숨겨진 꿀기능을 AI가 역으로 제안하게 만들면 업무 효율이 수십 배 올라갑니다.

## 🙋 자주 묻는 질문 (FAQ)

- **Q: 회사 내부 보안 때문에 엑셀 파일을 통째로 AI에 올릴 수가 없어요.**
  - A: 파일 전체를 업로드할 필요가 전혀 없습니다! 실제 데이터 대신 "홍길동, 서울시 강남구 (12345)"처럼 **가짜 예시 데이터 2~3줄**만 만들어서 프롬프트에 텍스트로 넣어주세요. AI는 패턴만 파악하면 완벽한 수식을 만들어 줍니다.

- **Q: 함수 수식을 복사해서 붙여넣었는데 에러가 납니다.**
  - A: 엑셀의 참조 셀 위치(예: A2)가 본인의 시트와 다를 수 있습니다. 프롬프트 결과창에서 A2를 참조하라고 했다면, 내 시트의 데이터 시작 위치가 어디인지 꼭 확인하세요. 계속 에러가 난다면 AI에게 "붙여넣었더니 #VALUE! 에러가 나. 뭐가 문제일까?"라고 에러 메시지 그대로 다시 질문하면 즉각 수정해 줍니다.

## 🧬 프롬프트 해부 (Why it works?)

1. **상황과 예시 제공 (Context):** `[서울시 강남구 테헤란로 123 (06234)]`라는 구체적인 입력/출력 예시를 제공하여 AI가 패턴을 정확히 인식하도록 돕습니다.
2. **구체적인 대안 요구 (Task):** 무조건 어려운 함수를 알려달라고 하지 않고, 내장 기능(빠른 채우기)이나 함수 중 '가장 효율적인 방법'을 먼저 판단하도록 지시했습니다.
3. **오류 방지 설계 (Warning):** `IFERROR`를 언급하여, 실무 데이터 특유의 불규칙성 때문에 수식이 깨지는 현상을 미연에 방지합니다.

## 📊 증명: Before & After

### ❌ Before (입력)

```text
"서울시 강남구 역삼동 (12345)" 이런 식인데 괄호 안 숫자만 빼고 싶어.
```

### ✅ After (결과)

```text
가장 빠르고 강력한 방법은 엑셀의 '빠른 채우기(Flash Fill)' 기능을 사용하는 것입니다.

[방법 1: 빠른 채우기 (가장 추천)]
1. B2 셀(주소)에 '서울시 강남구 테헤란로 123'을 직접 타이핑하고 Enter를 누릅니다.
2. C2 셀(우편번호)에 '06234'를 직접 타이핑하고 Enter를 누릅니다.
3. B3 셀을 클릭하고 단축키 `Ctrl + E`를 누르면 아래까지 주소가 자동 분리됩니다.
4. C3 셀을 클릭하고 단축키 `Ctrl + E`를 누르면 우편번호만 쏙 뽑혀 나옵니다.

[방법 2: 함수 사용 (자동화를 원할 때)]
데이터의 규칙성이 떨어져 함수가 필요하다면 아래 수식을 B2, C2에 복사해 넣으세요.
(괄호가 없는 예외 상황까지 고려한 수식입니다.)

- 주소 (B열): `=IFERROR(TRIM(LEFT(A2, FIND("(", A2)-1)), A2)`
- 우편번호 (C열): `=IFERROR(MID(A2, FIND("(", A2)+1, FIND(")", A2)-FIND("(", A2)-1), "")`
```
