## 📊 検証：Before & After

### ❌ Before（入力：非効率なPandasコード）

```python
# O(n^2)の非効率なデータ処理（実行時間: 4.2秒）
import pandas as pd
import numpy as np

df = pd.DataFrame({'A': np.random.rand(100000), 'B': np.random.rand(100000)})
result = []
for index, row in df.iterrows():
    result.append(row['A'] * row['B'] if row['A'] > 0.5 else row['B'] / 2)
df['Result'] = result
```

### ✅ After（結果：Gemini 3 Proによるベクトル化）

```python
# Vectorizationを適用した最適化コード（実行時間: 0.08秒 - 52倍高速化）
import pandas as pd
import numpy as np

df = pd.DataFrame({'A': np.random.rand(100000), 'B': np.random.rand(100000)})

# np.whereを用いたベクトル化演算
df['Result'] = np.where(df['A'] > 0.5, df['A'] * df['B'], df['B'] / 2)

# 【解説】iterrows()による行単位のループ処理はPythonのオーバーヘッドを伴いますが、
# NumPyのベクトル化された演算を使用することで、C言語レベルの連続したメモリブロックで
# 処理が実行され、劇的なパフォーマンス向上が見込めます。
```
