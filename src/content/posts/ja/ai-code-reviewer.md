## 📊 証明: Before & After

### ❌ Before (入力コード)

```python
def s(x):
    return x * x
```

### ✅ After (レビュー結果)

**👍 称賛する点:** 関数が簡潔で目的が明確です。

**🛠 改善する点:**

1. **ネーミング:** 関数名`s`と引数`x`は何の情報も与えません。`square`や`calculate_square`が良いでしょう。
2. **型ヒント:** Pythonの型ヒントを使用して入力と出力が数字であることを明示すると良いでしょう。
3. **Docstring:** 関数説明が抜けています。

**🤔 質問:**
もし入力値として数字ではなく文字列が入ってきたらどうなりますか？エラー処理を考慮してみましたか？

**✨ Refactoring:**

```python
def calculate_square(number: float) -> float:
    """与えられた数字の二乗を返します。"""
    return number ** 2
```
