## 🚀 The Solution: "SVG Mastermind Prompt"

### 🥉 Basic Version

Use this prompt when you need a quick, simple geometric shape or a minimalist icon.

> **Role:** You are an expert SVG developer and vector artist.
> 
> **Task:** Generate clean, valid, and responsive SVG code for a `[pelican riding a bicycle]`. Use semantic tags and keep the code completely efficient.

### 🥇 Pro Version

Use this advanced prompt when you require detailed, optimized, and visually stunning vector illustrations tailored for modern web projects.

> **Role:** You are a World-Class UI/UX Designer and Senior SVG Engineer. Your expertise lies in creating visually striking, mathematically perfect, and highly optimized vector graphics entirely through code.
>
> **Context:**
>
> - Background: I need a custom vector illustration for a modern web project, but I refuse to rely on generic stock assets or heavy raster image files.
> - Goal: Generate a complete, ready-to-use SVG file based strictly on my exact description.
>
> **Task:**
>
> 1. Write the complete, raw SVG code for: `[A detailed, stylized pelican riding a vintage bicycle]`.
> 2. Ensure the SVG is fully responsive (utilize a proper `viewBox`).
> 3. Use a cohesive, modern color palette (e.g., `#00f3ff` for neon accents against a dark theme).
> 4. Group related elements logically using `<g>` tags equipped with descriptive `id` or `class` attributes.
> 5. Do not include external dependencies or base64 images; rely solely on standard SVG paths, shapes, and gradients.
>
> **Constraints:**
>
> - Output ONLY the raw SVG code inside a single markdown code block. Do not include conversational filler, greetings, or explanations.
>
> **Warning:**
>
> - Ensure the XML is 100% valid. Unclosed tags or malformed paths will fatally break the render. Do not hallucinate unsupported SVG properties.

- **Q: Can I use these AI-generated SVGs commercially?**
  - A: Generally, yes. AI-generated raw code doesn't typically carry traditional stock license fees, making it perfectly safe for commercial implementation. However, you should always verify the specific terms of service of the AI platform you are currently using.

- **Q: The resulting SVG looks a bit abstract or flat. How do I make it more realistic?**
  - A: Vector generation via code naturally leans toward flat, minimalist design. To dramatically improve depth, modify the prompt to explicitly ask the AI to "incorporate intricate `<linearGradient>` shading, `<defs>` for dynamic shadows, and highly detailed `<path>` definitions."

1. **Dual Persona Formatting:** Combining the "UI/UX Designer" and "Senior SVG Engineer" roles ensures that the final output is both aesthetically pleasing and technically flawless.

3. **Structural Directives:** Explicitly requesting `<g>` tags and a proper `viewBox` forces the AI to write professional, highly maintainable, and responsive code rather than a messy, unorganized dump of raw paths.
