## 📊 증명: Before & After

### ❌ Before (입력 CSV)

```csv
이름,나이,직업
홍길동,30,개발자
김철수,25,디자이너
```

### ✅ After (결과 - JSON + 필드명 변경)

**요청:** CSV를 JSON으로 바꾸되, 영어 필드명으로 변경해줘.

```json
[
  {
    "name": "홍길동",
    "age": 30,
    "job": "개발자"
  },
  {
    "name": "김철수",
    "age": 25,
    "job": "디자이너"
  }
]
```

### ✅ After (결과 - SQL Insert문)

**요청:** MySQL `users` 테이블에 넣을 수 있는 INSERT 문으로 바꿔줘.

```sql
INSERT INTO users (name, age, job) VALUES
('홍길동', 30, '개발자'),
('김철수', 25, '디자이너');
```
