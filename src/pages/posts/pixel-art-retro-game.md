# 📝 도트 장인: Pixel Art

- **🎯 추천 대상:** 누구나
- **⏱️ 소요 시간:** 5분
- **🤖 추천 모델:** 모든 AI 모델

- **📊 난이도:** ⭐⭐☆☆☆
- **⚡️ 효과성:** ⭐⭐⭐⭐⭐
- **🛠️ 활용도:** ⭐⭐⭐⭐☆

_"슈퍼 마리오나 포켓몬스터 골드 버전 같은 도트 그래픽이 필요해."_

**Pixel Art**(픽셀 아트)는 점(픽셀) 하나하나를 찍어 만든 디지털 아트의 시초입니다. 해상도의 제약 속에서 피어나는 상상력과 특유의 귀여움 덕분에 지금도 사랑받는 스타일입니다.

---

## ⚡️ 3줄 요약 (TL;DR)

1. `Pixel art`, `8-bit` (단순), `16-bit` (디테일) 키워드를 구분해 쓰세요.
2. `Isometric`과 결합하면 고전 RPG 게임 맵을 만들 수 있습니다.
3. 미드저니에서는 `--v 6.0`보다 `--niji 6`가 픽셀 느낌을 잘 살리기도 합니다.

## 🚀 해결책: "Retro Game Prompt"

### 🥉 Basic Version (기본형)

빠르게 결과만 필요할 때 사용하세요.

> **역할:** 너는 전문가야.
> **요청:** 고전 게임의 향수를 불러일으키는 8비트, 16비트 픽셀 아트 프롬프트해줘.

<br>

### 🥇 Pro Version (전문가형)

디테일한 퀄리티가 필요할 때 사용하세요.

아래 **PROMPT** 내용을 복사해서 사용하세요.

> **Prompt:**
>
> **Subject:** A brave knight fighting a green dragon in a dungeon.
>
> **Style:** Pixel art, 16-bit graphics, SNES style, retro game asset, dithering effect, vibrant colors.
>
> **Detail:** Clean pixels, no anti-aliasing, side scrolling view.
>
> **Parameters (Midjourney):** --ar 4:3 --v 6.0 --stop 90
