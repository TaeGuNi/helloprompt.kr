## 📊 証明: Before & After

### ❌ Before (状況)

**エラー:** `TypeError: Cannot read properties of undefined (reading 'map')`
**コード:**

```javascript
function renderList(data) {
  return data.users.map((user) => `<li>${user.name}</li>`);
}
```

### ✅ After (結果)

**分析:** `data.users`が存在しないか(undefined)、配列でない状態で`map`関数を呼び出しました。APIレスポンスがまだ到着していないか、形式が変更された可能性が高いです。

**修正コード:**

```javascript
function renderList(data) {
  // オプショナルチェイニングとデフォルト値を使用
  const users = data?.users || [];

  if (users.length === 0) return "<p>ユーザーがいません。</p>";
  return users.map((user) => `<li>${user.name}</li>`);
}
```
