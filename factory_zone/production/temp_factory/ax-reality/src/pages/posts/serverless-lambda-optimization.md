# ⚡️ 서버리스(Serverless) 최적화: AWS Lambda 콜드 스타트 해결법

- **🎯 추천 대상:** "처음 접속할 때만 느려요"라는 VOC에 시달리는 개발자, 서버 관리하기 귀찮아서 람다로 도망친 사람
- **⏱️ 소요 시간:** 10분 (설정 변경)
- **🤖 추천 모델:** Claude 3.5 Sonnet (인프라 최적화)

- ⭐ **난이도:** ⭐⭐⭐☆☆
- ⚡️ **효과성:** ⭐⭐⭐⭐⭐
- 🚀 **활용도:** ⭐⭐⭐⭐⭐

> _"서버리스라며? 왜 이렇게 느려?"_

서버가 없다는 건, **"필요할 때만 켠다"**는 뜻입니다. 꺼져 있던 서버를 켜는 시간, 그게 바로 **콜드 스타트(Cold Start)**입니다. 3초가 걸릴 수도 있죠. 이걸 0.1초로 줄이는 마법, AI에게 물어보세요.

---

## ⚡️ 3줄 요약 (TL;DR)

1.  **메모리:** 메모리를 늘리면 CPU도 같이 늘어서 빨라진다.
2.  **언어:** 자바(Java)보다 **Go**나 **Node.js**가 시동이 빠르다.
3.  **Pro:** **Provisioned Concurrency(미리 켜두기)**를 쓴다.

---

## 🚀 해결책: "Lambda Tuner Prompt"

### 🥉 Basic Version (메모리 튜닝)

돈으로 해결하기.

> **상황:** "지금 128MB 람다를 쓰고 있는데, 첫 요청이 2초나 걸려."
> **요청:** "메모리를 1GB로 올리면 콜드 스타트가 얼마나 줄어들까? 비용은 얼마나 늘어나? **AWS Power Tuning** 도구를 써서 최적의 가성비 구간을 찾는 법을 알려줘."

<br>

### 🥇 Pro Version (코드 레벨 최적화 & 스냅샷)

근본적인 다이어트.

> **역할 (Role):** 너는 서버리스 성능 최적화 전문가야.
>
> **내 환경 (Env):** Node.js 20, AWS SDK v3 사용.
>
> **요청 (Task):**
>
> 1.  **번들 사이즈:** 코드가 무거우면 로딩이 느려. `esbuild`로 불필요한 라이브러리를 쳐내고(Tree Shaking) 파일을 하나로 합치는 설정 코드를 줘.
> 2.  **SDK 최적화:** `aws-sdk` 전체를 불러오지 않고, 필요한 모듈(예: `S3Client`)만 가볍게 불러오는 코드로 바꿔줘.
> 3.  **SnapStart:** 자바(Java) 람다의 구세주인 **SnapStart** 기능을 활성화하는 방법과 원리를 설명해줘. (메모리 상태를 저장해두고 복원하는 기술)

## 💡 작성자 코멘트 (Insight)

가장 확실한 방법은 **'Provisioned Concurrency'**입니다.
"서버 1개는 항상 켜놔"라고 돈을 내고 예약하는 겁니다. (서버리스의 취지와 좀 안 맞지만...)
트래픽이 몰리는 시간대(9~6시)에만 오토 스케일링으로 예약 걸어두면 비용과 성능을 다 잡을 수 있습니다.

## 🙋 자주 묻는 질문 (FAQ)

- **Q: VPC 안에 넣으면 느려지나요?**
  - A: 옛날엔 그랬습니다(ENI 생성 시간). 하지만 최근 AWS 업데이트로 거의 차이 없어졌습니다. 안심하고 넣으세요.

- **Q: 람다 말고 다른 건요?**
  - A: **CloudFront Functions**나 **Lambda@Edge**는 엣지(Edge)에서 실행돼서 훨씬 빠릅니다. 가벼운 로직은 이쪽으로 빼세요.

---

## 🧬 프롬프트 해부 (Why it works?)

1.  **번들 사이즈:** 콜드 스타트의 주범은 '코드 다운로드 시간'입니다. AI에게 "트리 쉐이킹(Tree Shaking)"을 요구해서 뚱뚱한 코드를 다이어트시키는 게 핵심입니다.
2.  **SnapStart:** 최신 기술(SnapStart)을 언급하면, AI가 단순 튜닝을 넘어선 고급 기능(CRaC 등)을 제안해 줍니다.

---

## 📊 증명: Before & After

### ❌ Before (통통한 람다)

`import AWS from 'aws-sdk'` (전체 로드) -> 용량 50MB -> 콜드 스타트 1.5초 🐢

### ✅ After (날씬한 람다)

`import { S3Client } from '@aws-sdk/client-s3'` (필요한 것만) -> 용량 1MB -> 콜드 스타트 **0.2초** 🚀
