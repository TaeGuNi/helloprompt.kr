---

## 📊 証明: Before & After

### ❌ Before (入力 - Python)

```python
result = [x * 2 for x in range(10) if x % 2 == 0]
```

### ✅ After (プロンプト結果)

**[要約]**
0から9までの数字のうち偶数だけを選び出し、その値を2倍にした後新しいリストに保存するコードです。

**[ライン・バイ・ライン分析]**

```python
# 0から9まで(range(10))数字を一つずつ取り出して変数xに入れます。
# もし(if) xを2で割った余りが0なら(偶数なら)、
# xに2を掛けた値をリストに入れます。
result = [x * 2 for x in range(10) if x % 2 == 0]
```

**[文法ポイント：リスト内包表記]**
Python特有の文法で、`for`繰り返し文と`if`条件文を一行で簡潔にリストを生成する時に使います。

- 構造: `[表現式 for 項目 in 反復可能オブジェクト if 条件]`
