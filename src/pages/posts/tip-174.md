# 📝 엑셀 민감도 분석: "만약에" 시나리오 완벽 대비

- **🎯 추천 대상:** "환율 오르면 어쩔 건데?"라는 임원 질문에 말문이 막히는 기획자
- **⏱️ 소요 시간:** 2시간 노가다 → 1분 컷
- **🤖 추천 모델:** ChatGPT-4, Excel Copilot

_"미래는 아무도 모릅니다. 하지만 모든 미래를 계산해둘 순 있습니다."_

사업 계획을 짤 때 가장 중요한 건 변수입니다. 환율, 원자재 가격, 판매량 등이 변할 때 영업이익이 어떻게 바뀌는지 보여주는 **'민감도 분석(Sensitivity Analysis)'** 테이블. 엑셀의 숨겨진 치트키 **'데이터 표(Data Table)'** 기능을 쓰면 수식 하나로 100가지 시나리오를 자동 생성할 수 있습니다.

---

## ⚡️ 3줄 요약 (TL;DR)

1. 변수(환율, 판매량)와 결과값(영업이익)의 관계 설정
2. '가상 분석 > 데이터 표' 기능을 활용한 매트릭스 생성
3. 두 가지 변수가 동시에 변할 때의 결과값 시뮬레이션

---

## 🚀 해결책: "시나리오 매트릭스 생성기"

아래 **PROMPT** 내용을 복사해서 사용하세요.

> **역할 (Role):** 너는 전략 기획실의 `[재무 모델링 전문가]`야.
>
> **상황 (Context):** 엑셀에서 **민감도 분석(Sensitivity Analysis)**을 하려고 해.
>
> - **목표:** '영업이익' 변화 관찰
> - **변수 1 (행):** 환율 (1,200원 ~ 1,400원, 50원 단위)
> - **변수 2 (열):** 판매량 (1만 개 ~ 5만 개, 1만 개 단위)
>
> **요청 (Task):**
>
> 1. 엑셀의 **'데이터 표(Data Table)'** 기능을 사용하는 순서를 단계별로 알려줘.
> 2. 행 입력 셀과 열 입력 셀을 어디에 연결해야 하는지 헷갈리지 않게 설명해줘.
> 3. 완성된 표를 보고 "환율이 오르면 이익이 급감하므로 헷징이 필요하다" 같은 인사이트를 도출하는 팁을 줘.
>
> **제약사항 (Constraints):**
>
> - 메뉴 경로(데이터 탭 > 가상 분석 > 데이터 표)를 명시해줘.

---

## 🧬 프롬프트 해부 (Why it works?)

'데이터 표' 기능은 강력하지만 설정법이 매우 까다롭습니다. 특히 '행 입력 셀'과 '열 입력 셀'을 반대로 넣으면 결과가 엉망이 됩니다. 이 프롬프트는 그 **'연결 포인트'**를 정확히 짚어달라고 요청하여, 사용자가 한 번에 성공할 수 있도록 가이드합니다.

---

## 📊 증명: Before & After

### ❌ Before (입력)

```text
엑셀로 환율 변동 시나리오 어떻게 만들어?
```

### ✅ After (결과)

```text
### 🌪️ 엑셀 민감도 분석 3단계 가이드

**1. 판 깔기 (매트릭스 구조)**
- A열(세로)에는 **환율** 시나리오(1200, 1250...)를 입력하세요.
- 1행(가로)에는 **판매량** 시나리오(1만, 2만...)를 입력하세요.
- **핵심:** A열과 1행이 만나는 **모서리(A1)** 셀에 반드시 **'영업이익 계산 수식(=C10)'**을 연결해야 합니다.

**2. 마법 부리기 (데이터 표 실행)**
- 표 전체를 드래그 선택 ➔ [데이터] 탭 ➔ [가상 분석] ➔ [데이터 표] 클릭.
- **행 입력 셀:** 원본 데이터의 **'판매량'** 셀 클릭 (가로축이니까!)
- **열 입력 셀:** 원본 데이터의 **'환율'** 셀 클릭 (세로축이니까!)

**3. 결과 확인**
- 확인을 누르는 순간, 표 안의 빈칸들이 수십 개의 영업이익 예상치로 꽉 찹니다.
- **💡 팁:** 조건부 서식(색조)을 입히세요. 이익이 적자(빨강)로 돌아서는 '데드라인'이 한눈에 보입니다.
```

## 🚨 트러블 슈팅 (안 될 땐 이렇게!)

**Q. 값이 안 바뀌고 다 똑같이 나와요.**
A. 엑셀 계산 옵션이 '수동'으로 되어 있을 확률이 높습니다. [수식] 탭 ➔ [계산 옵션] ➔ **'자동'**으로 체크되어 있는지 확인하세요. 데이터 표는 리소스를 많이 먹어서 가끔 자동으로 안 바뀔 때가 있습니다.
