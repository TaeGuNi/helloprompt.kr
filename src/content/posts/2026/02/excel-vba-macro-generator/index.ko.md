# 📝 반복되는 엑셀 노가다, 매크로(VBA)로 자동화하기

- **🎯 추천 대상:** 매일 같은 엑셀 데이터를 복사/붙여넣기 하는 직장인, 마케터, 영업 관리자
- **⏱️ 소요 시간:** 5분 투자로 매일 30분 단축
- **🤖 추천 모델:** ChatGPT (GPT-4o), Claude 3.5 Sonnet (코드 생성에 탁월함)

- ⭐ **난이도:** ⭐⭐☆☆☆ (코딩 지식 불필요)
- ⚡️ **효과성:** ⭐⭐⭐⭐⭐
- 🚀 **활용도:** ⭐⭐⭐⭐⭐

> _"아직도 매일 아침 '어제자 데이터'를 복사해서 '오늘자 시트'에 붙여넣고, 빈칸 지우고, 테두리 그리는 작업을 손으로 하고 계신가요?"_

매일 반복되는 데이터 취합, 서식 변경, 필터링 작업은 직장인의 소중한 오전 시간을 갉아먹는 주범입니다. '매크로(VBA)를 쓰면 편하다'는 건 누구나 알지만, 막상 코딩을 배우려니 막막해서 포기하셨나요?

이제 코딩을 배울 필요가 없습니다. AI에게 **"내가 지금 엑셀에서 어떤 행동을 반복하고 있는지"**만 명확하게 설명하면, 복붙만 하면 끝나는 완벽한 매크로 코드를 짜주니까요.

---

## ⚡️ 3줄 요약 (TL;DR)

1. 코딩 지식이 전혀 없어도 AI를 통해 엑셀 매크로(VBA) 코드를 완벽하게 생성할 수 있습니다.
2. 실행 속도 최적화, 에러 방지 등 전문가 수준의 코드를 얻어내는 최적의 프롬프트를 제공합니다.
3. 매일 30분씩 걸리던 단순 반복 노가다를 버튼 클릭 한 번(1초)으로 끝내고 칼퇴하세요.

## 🚀 해결책: "VBA 코딩 마스터"

### 🥉 Basic Version (기본형)

복잡한 조건 없이, 아주 단순한 작업(예: 특정 열 삭제, 시트 통합 등)을 자동화하고 싶을 때 사용하세요.

> **역할:** 너는 20년 차 엑셀 VBA 마스터 개발자야.
> **요청:** 내가 설명하는 반복 작업을 자동화하는 VBA 코드를 작성해줘.
> **작업 내용:** `[여기에 엑셀에서 반복하는 작업을 자연어로 적으세요. 예: 시트1의 A열 빈칸을 모두 지우고, B열 기준으로 오름차순 정렬해줘]`
> **조건:** 코드를 엑셀에 어떻게 적용하고 실행하는지 초보자도 알기 쉽게 1번부터 순서대로 설명해.

<br>

### 🥇 Pro Version (전문가형)

수십 만 행의 대용량 데이터 처리, 여러 파일 취합, 복잡한 예외 처리 등 **실무에서 발생하는 변수까지 완벽하게 통제된 고품질 코드**가 필요할 때 사용하세요.

> **역할 (Role):** 너는 글로벌 대기업의 업무 프로세스를 자동화하는 '수석 엑셀 VBA 엔지니어'야. 코딩을 전혀 모르는 일반 사무직 직원을 위해 완벽하게 작동하는 코드를 작성해야 해.
>
> **상황 (Context):**
>
> - 현재 상태: `[예: 매일 아침 10개의 지점별 매출 엑셀 파일을 열어서 하나의 마스터 파일로 취합하고 있음]`
> - 해결하고 싶은 문제점: `[예: 파일명 규칙이 가끔 다르고, 중간에 빈 행이 섞여 있어서 수작업 시 에러가 잦음]`
>
> **요청 (Task):**
>
> 1. 위 상황을 100% 자동화하는 최적화된 VBA 매크로 코드를 작성해.
> 2. 작성된 코드의 핵심 로직이 무엇인지 주석(`'`)을 한국어로 상세히 달아줘.
> 3. 엑셀을 켜고 이 코드를 복사해서 단축키로 실행하기까지의 모든 과정을 메뉴 이름과 단축키(예: Alt+F11)를 포함하여 초보자 눈높이에서 단계별로 안내해.
>
> **제약사항 (Constraints):**
>
> - **최적화:** 대용량 데이터에서도 엑셀이 멈추지 않도록 `Application.ScreenUpdating = False`, `Application.Calculation = xlCalculationManual` 등의 속도 최적화 코드를 반드시 시작과 끝에 포함할 것.
> - **안정성:** 만약 지정한 파일이나 시트가 없을 경우, 매크로가 멈추는 대신 "해당 파일을 찾을 수 없습니다"라는 친절한 메시지 창(MsgBox)을 띄우고 안전하게 종료되는 에러 핸들링 코드를 넣을 것.
> - **출력 형식:** 코드는 반드시 마크다운 코드블럭(```vba) 안에 작성할 것.

## 💡 작성자 코멘트 (Insight)

이 프롬프트의 핵심은 **`제약사항(Constraints)`**에 있습니다. 일반적인 프롬프트로 VBA를 짜달라고 하면, AI는 딱 '기능이 돌아가기만 하는' 허술한 코드를 줍니다. 데이터가 10만 줄을 넘어가면 엑셀이 응답 없음으로 뻗어버리거나, 중간에 빈칸 하나만 있어도 런타임 에러를 뱉어버리죠.

Pro 버전의 프롬프트는 화면 업데이트 정지(`ScreenUpdating = False`)와 같은 현업 개발자들의 필수 최적화 기법과 에러 핸들링을 강제합니다. 이렇게 생성된 코드는 수십 메가바이트의 무거운 엑셀 파일에서도 버벅임 없이 1초 만에 작업을 끝내줍니다. 현업에서 정말 요긴하게 쓰이는 '방어적 프로그래밍'이 적용된 결과물을 얻을 수 있습니다.

## 🙋 자주 묻는 질문 (FAQ)

- **Q: 회사 보안 때문에 엑셀 파일 자체를 AI에 올릴 수가 없는데 어떡하나요?**
  - A: 파일은 절대 올리지 마세요! 대신 "A열에는 날짜, B열에는 상품명, C열에는 매출액이 있는 표에서..." 와 같이 데이터의 **구조(컬럼명)**와 **작업 순서**만 텍스트로 자세히 설명해주면 됩니다. 코드는 여러분의 PC 엑셀에서 로컬로 안전하게 돌아갑니다.

- **Q: 코드를 복사해서 넣었는데 컴파일 에러(Compile Error)가 납니다.**
  - A: AI에게 발생한 에러 메시지를 그대로 복사해서 "이런 에러가 나는데 어떻게 수정해?"라고 다시 물어보세요(Troubleshooting). 보통 셀 참조 범위나 변수 타입이 안 맞아서 생기는 문제라, AI가 즉시 원인을 파악하고 수정된 코드를 다시 줍니다.

- **Q: 구글 스프레드시트에서도 쓸 수 있나요?**
  - A: VBA는 마이크로소프트 엑셀 전용 언어입니다. 구글 스프레드시트를 쓰신다면 프롬프트의 역할(Role)을 "VBA 개발자"에서 **"Google Apps Script(GAS) 개발자"**로만 바꿔서 똑같이 질문해 보세요. 자바스크립트 기반의 완벽한 GAS 코드를 짜줍니다.

## 🧬 프롬프트 해부 (Why it works?)

1. **페르소나 최적화:** '수석 VBA 엔지니어'라는 구체적인 역할을 부여하여, 단순히 코드를 짜는 것을 넘어 초보자가 실행 가능한 수준까지 배려하도록 유도했습니다.
2. **에러 핸들링 강제:** 실무 엑셀 환경은 변수가 많습니다. 예외 상황(파일 누락, 빈 행 등)에 대비한 에러 처리 지시를 통해 코드의 견고성(Robustness)을 극대화했습니다.
3. **성능 최적화 지시:** `ScreenUpdating = False` 등 현업 최적화 기법을 명시하여 대용량 엑셀 파일에서도 시스템이 다운되지 않고 빠르게 동작하도록 안전장치를 마련했습니다.

## 📊 증명: Before & After

### ❌ Before (단순한 지시)

```text
시트1의 데이터를 시트2로 옮기고 A열 빈칸은 지워줘.
```

> **결과:** 속도 최적화가 안 되어 있어 데이터가 많으면 화면이 껌뻑거리며 느리게 작동하고, 만약 지정한 시트가 없으면 곧바로 런타임 에러 창을 띄우며 엑셀이 멈춤.

### ✅ After (Pro 프롬프트 적용 결과)

```text
Sub OptimizeAndCopyData()
    ' 속도 최적화 시작
    Application.ScreenUpdating = False
    Application.Calculation = xlCalculationManual

    On Error GoTo ErrorHandler ' 에러 핸들링 방어 코드

    ' (중략: 데이터 복사 및 빈칸 삭제 핵심 로직)

    MsgBox "작업이 성공적으로 완료되었습니다!", vbInformation

ExitSub:
    ' 최적화 원상복구 (매우 중요)
    Application.ScreenUpdating = True
    Application.Calculation = xlCalculationAutomatic
    Exit Sub

ErrorHandler:
    MsgBox "오류가 발생했습니다. 대상 시트가 있는지 확인해주세요.", vbCritical
    Resume ExitSub
End Sub
```

> **결과:** 화면 깜빡임 없이 즉각 처리되며, 완료/에러 상황에 대해 깔끔한 메시지 창을 띄워주는 전문가 수준의 매크로 생성.
