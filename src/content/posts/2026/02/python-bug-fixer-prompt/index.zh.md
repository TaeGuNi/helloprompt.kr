- **🎯 推荐对象：** 初中级开发者、数据分析师、经常与Python打交道的职场人
- **⏱️ 预计耗时：** 30分钟 → 缩短至 10秒
- **🤖 推荐模型：** 任何对话型 AI (ChatGPT, Claude 3.5 Sonnet, Gemini Advanced 等)

- ⭐ **难度：** ⭐☆☆☆☆
- ⚡️ **效果：** ⭐⭐⭐⭐⭐
- 🚀 **实用度：** ⭐⭐⭐⭐⭐

> _"明明代码没改什么，怎么突然就跑不通了？面对满屏的红色报错信息，你的血压是不是又飙升了？"_

无论你是刚入门的编程新手，还是日常处理数据的分析师，Python 报错（Error）绝对是开发过程中最让人抓狂的环节。与其在 Stack Overflow 上大海捞针般地搜索报错信息，不如直接让 AI 成为你的专属高级工程师，一针见血地指出问题所在并提供修复方案。

## ⚡️ 3句话总结 (TL;DR)

1. **精准定位：** 直接将报错信息连同代码丢给 AI，瞬间找到 Bug 根源。
2. **结构化输出：** 强制 AI 按照“原因分析-修复代码-预防建议”的结构输出，避免废话。
3. **零门槛上手：** 无论是语法错误（SyntaxError）还是逻辑漏洞，都能一键套用，节约 90% 的调试时间。

>
> **代码：**
> `[粘贴你的代码]`

### 🥇 Pro Version (专家版)

当你不仅想解决问题，还想理解底层逻辑，提升自身编程能力时使用。

> **角色 (Role)：** 你是一位拥有 15 年经验的顶级 Python 架构师，擅长代码审查（Code Review）和性能优化。
>
> **背景 (Context)：**
>
> - 当前状况：我在运行一段 Python 代码时遇到了难以解决的 Bug。

>
> **任务 (Task)：**
> 请仔细阅读我提供的代码和报错信息，并按照以下步骤执行：
>
> 1. **Bug 诊断：** 用最通俗易懂的语言解释导致这个错误的根本原因是什么。
> 2. **代码修复：** 提供修改后的完整且可运行代码。必须对修改的部分添加详细的中文注释。
> 3. **防范建议：** 提供 1-2 条关于如何避免此类 Bug 的最佳实践建议。
>
> **输入变量：**
>
> - **[我的代码]：**
> - **[报错信息 / 终端输出]：**
>
> **约束条件 (Constraints)：**
>
> - 不要给出模棱两可的答案。如果报错信息不足以定位问题，请直接告诉我还需要补充什么信息（例如变量状态、上下文环境）。
> - 修复后的代码必须符合 PEP 8 规范。

> **注意事项 (Warning)：**
>
> - 如果代码本身没有语法错误，而是逻辑错误或环境依赖问题，请明确指出。禁止胡乱猜测（防止幻觉）。

## 🙋 常见问题 (FAQ)

- **Q: 如果报错信息太长，超出了 AI 的字数限制怎么办？**
  - A: 截取最核心的部分。通常来说，Python 的 Traceback 报错信息中，**最后 3-5 行**才是最关键的错误说明。你只需要提供你的核心业务代码以及最后几行的报错日志即可。

- **Q: AI 给出的修复代码运行后又报了新的错？**
  - A: 这是一个常见的迭代过程。你可以直接回复 AI：“你的代码报了新的错误：[新报错信息]。请结合上下文重新思考并修复。” 赋予了架构师角色的 AI 具备极强的上下文修正能力。

- **Q: 这个提示词适用于其他编程语言吗？**
  - A: 完全可以！只需要将提示词中的 "Python" 替换为 "JavaScript"、"Java" 或 "C++"，同时将 PEP 8 替换为对应语言的编码规范（如 ESLint），同样能获得极佳的调试效果。

## 📊 效果对比：Before & After

### ❌ Before (传统的低效提问)

```text
这代码怎么报错了？帮我看看。
TypeError: can only concatenate str (not "int") to str

age = 25
print("我今年" + age + "岁")
```

### ✅ After (使用 Pro Version 提示词后的极效输出)

````text
**Bug 诊断：**
根本原因在于 Python 是强类型语言，不允许直接将字符串（str）和整数（int）进行拼接。代码中试图将整数类型的 `age` 直接与字符串相加，触发了 `TypeError`。

**代码修复：**
```python
age = 25
# 修复：使用 f-string (格式化字符串字面值)，这是 Python 3.6+ 推荐的高效拼接方式
print(f"我今年{age}岁")

# 备选方案：如果必须使用加号拼接，需要先将整数转换为字符串
# print("我今年" + str(age) + "岁")
````

**防范建议：**
在进行字符串拼接时，强烈建议优先使用 `f-string`（在字符串前加 `f`），它不仅可读性极佳，还能自动处理绝大多数的数据类型转换，有效避免此类 `TypeError`。

