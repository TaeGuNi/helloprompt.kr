### ❌ Antes (Flujo de trabajo clásico y limitado)

```python
    tf.keras.layers.Dense(256, activation='relu', input_shape=(100,)),
    tf.keras.layers.Dense(128, activation='relu'),
    tf.keras.layers.Dense(64, activation='relu'), # Cuello de botella de optimización clásico
    tf.keras.layers.Dense(10, activation='softmax')
])
```

### ✅ Después (Arquitectura híbrida generada por el prompt)

```python
import tensorflow_quantum as tfq
import cirq
import tensorflow as tf
import sympy

# 1. Definir los Qubits (El circuito cuántico)
qubits = [cirq.GridQubit(0, i) for i in range(4)]

# 2. Construir el Circuito Cuántico Parametrizado (PQC)
circuit = cirq.Circuit()
symbols = sympy.symbols('theta0:4')
for i, qubit in enumerate(qubits):
    circuit.append(cirq.ry(symbols[i])(qubit))

# 3. Integración en el modelo clásico de Keras (El Pipeline Híbrido)
model = tf.keras.Sequential([
    tf.keras.layers.Dense(64, activation='relu', input_shape=(100,)),
    # Reemplazo de la capa oculta por una capa cuántica para mayor expresividad
    tfq.layers.PQC(circuit, cirq.Z(qubits[0])),
    tf.keras.layers.Dense(10, activation='softmax')
])

# El entrenamiento sigue utilizando optimizadores clásicos (ej. Adam)
model.compile(optimizer=tf.keras.optimizers.Adam(learning_rate=0.01),
              loss='categorical_crossentropy',
              metrics=['accuracy'])
```
