- **⏱️ 所需时间：** 5分钟（文本提取与 AI 深度分析）
- **🤖 推荐模型：** ChatGPT Plus（需开启 GPT-4o 及高级数据分析功能）

- ⭐ **难度级别：** ⭐⭐☆☆☆
- ⚡️ **效果显著：** ⭐⭐⭐⭐⭐
- 🚀 **实用程度：** ⭐⭐⭐⭐⭐

记录唯有经过分析，才能沉淀为资产，否则不过是锁在抽屉里的情绪包袱。现在，试着将**整月的日记数据**交给 AI 吧。借助强大的文本挖掘（Text Mining）技术，AI 能够精准勾勒出隐藏在字里行间的真实情绪曲线，并将潜意识中反复横跳的词汇转化为直观的视觉图表。是时候去破解那些连你自己都未曾察觉的心理算法了。

## ⚡️ 三行总结 (TL;DR)

1. 从你常用的日记 App 或 Notion 中，将过去一个月的记录导出为 `txt` 或 `csv` 格式文件。
2. 将文件一键上传至 ChatGPT，并发送**数据可视化与情感分析**的专属提示词。
3. 查阅 AI 自动生成的情绪波动折线图及压力源诊断报告，完成一场高效、深度的自我复盘。

## 🚀 解决方案："Mind Tracker" 情绪追踪分析提示词

### 🥉 Basic Version (基础版)

当你只想快速摸清潜意识里的核心关注点与当前情绪底色时，请使用此版本。

> **角色：** 你是一位资深的数据分析师兼心理咨询师。
> **请求：** 请从我上传的近一周日记文本中，提取出出现频率最高的前 5 个名词和形容词。基于这些高频词汇，请用一句话精准概括我当下的核心心理状态。

### 🥇 Pro Version (专业版)

此为进阶的专业版提示词。它不仅能提供精准总结，还能通过科学的数据可视化手段，深度剖析你每日情绪的波动轨迹及其潜在诱因。

> **角色 (Role)：** 你是一位拥有 10 年临床经验的数据科学家兼心理学家。你极擅长通过深度剖析用户的文本数据，洞察其底层的心理模式，并以专业图表进行可视化呈现。
> 
> **背景 (Context)：**
> 
> - 背景：我虽有每日记日记的习惯，但近期情绪起伏剧烈。我渴望借助客观的数据分析，精准定位情绪波动的幕后黑手。
> - 目标：基于我提供的日记数据，绘制出整个月的心理波动曲线图，并深度挖掘诱发压力的根本原因与情绪自愈的潜在规律。
> 
> **任务 (Task)：**
> 
> 1. **情绪量化：** 深度读取我提供的日记文本，将每日的情绪状态进行量化打分，分值范围设定为 -10（极度抑郁/压力爆表）至 +10（极致幸福/成就感爆棚）。
> 2. **数据可视化（运行 Python）：** 请直接调用 Python 环境，使用**折线图 (Line Chart)** 将每日的情绪轨迹直观地绘制出来。
>    - x 轴设定为“日期”，y 轴设定为“情绪分数”。
>    - 请使用 `matplotlib` 或 `seaborn` 库，渲染出结构清晰、视觉美观的专业图表。
> 3. **深层诱因剖析：** 重点提取情绪分数跌至最低点（谷底）与飙至最高点（巅峰）对应的日记文本，深度解析究竟是哪些具体事件或核心关键词，对我的情绪产生了最直接、最具破坏性或治愈性的影响。
> 4. **行为处方：** 为了赋能下个月的心态管理，请基于上述数据洞察，为我开具一份“行为处方”：包含 3 个我应当刻意“增加”的微习惯，以及 3 个我必须立刻“戒断”的毒性行为。
> 
> **约束条件 (Constraints)：**
> 
> - 所有的分析结论必须**严格锚定**于我提供的文本数据，严禁任何形式的 AI 幻觉或凭空捏造。
> - 在渲染可视化图表时，请务必提前注入支持中文字体的配置（例如：`plt.rcParams['font.sans-serif'] = ['SimHei']`），彻底杜绝中文乱码（方块字）现象。
> - 最终交付物必须采用结构化的 Markdown 格式排版，呈现出专业体检报告般的严谨质感。
> 
> **警告 (Warning)：**
> 
> - 绝对禁止对我的日记内容进行任何形式的道德审判或价值观说教。你必须始终恪守客观中立、且不失同理心的专业边界。

## 💡 作者深度洞察 (Writer's Insight)

这套提示词的核心杀伤力在于，它用冷峻的数据将我们模糊的**“元认知 (Meta-cognition)”**彻底具象化了。我也曾深陷职业倦怠的泥沼，每天只能在日记里无力地敲下“心力交瘁、想裸辞”的废话。然而，当我将这套提示词作用于整月的日记后，一份令人毛骨悚然的真相浮出水面——所有情绪坠入冰点的日子，背后都潜伏着同一个致命触发链条：“连续两天睡眠不足”叠加“与某位特定高管的汇报会议”。

如果你恰好没有记日记的习惯，强烈建议你导出**“文件传输助手”或“单人微信群”**的碎碎念记录来做平替。更硬核的玩法是，直接喂给 AI 你在办公软件（如飞书、钉钉、Slack）中的发言记录。你将赤裸裸地直面自己：究竟在哪个时间段最具攻击性？又是在推进哪条业务线时，潜意识里的积极词汇密度最高？记住，人类的记忆极易被情绪篡改，但底层数据，从不撒谎。

## 🙋 常见问题解答 (FAQ)

- **Q: 如果我用的是纸质笔记本，那些纯手写的日记也能分析吗？**
  - A: 完全没问题。直接用手机拍下内页，将高清图片甩给 ChatGPT 即可。得益于 GPT-4o 堪称逆天的 OCR（光学字符识别）能力，哪怕是“医生体”般潦草的字迹，也能被精准扒成纯文本，并无缝接入后续的深度分析。

- **Q: 把极其私密的日记直接喂给 AI，会不会有严重的隐私泄露风险？**
  - A: 你的安全意识非常到位。在投喂任何隐私数据前，请务必潜入 ChatGPT 的「设置 (Settings)」 > 「数据控制 (Data Controls)」面板，**果断关闭“使用我的内容改进模型 (Improve the model for everyone)”**的开关。为了守住绝对的安全底线，强烈建议在上传文件前，动用一键替换功能，将文中的真实姓名、公司名称等高危专有名词，统一脱敏为“A项目”、“B同事”。

- **Q: AI 渲染出的图表里，中文字符集体变成了方块（□）乱码，怎么破？**
  - A: 这是 ChatGPT 在沙盒内调用 Python 绘图库时的“祖传 Bug”。解法极其简单，只需在提示词中强行注入一句指令：“*在渲染图表时，必须强制配置中文字体（如 `plt.rcParams['font.sans-serif'] = ['SimHei']`），确保全量中文绝不乱码*”。别担心，我们在 Pro 版提示词里，已经把这行防御代码提前焊死了。

## 🧬 提示词解剖 (Why it works?)

1. **强制唤醒 Python 引擎 (`数据可视化（运行 Python）`)：** 我们绝非在陪 AI 玩轻量级的文字游戏，而是通过硬核指令，强行拉起其内置的 Python 数据分析沙盒（Advanced Data Analysis）。相较于长篇大论的八股文报告，一张起伏跌宕的情绪折线图，能瞬间对你的自我认知造成降维打击。
2. **主观数据定量降维 (`量化打分`)：** 强行将充斥着主观宣泄的情绪废料，清洗并转化为算法可读、可计算的客观量化指标。这把衡量情绪的刻度尺越是精密，最终跑出的曲线图就越具备不可辩驳的公信力。
3. **打造闭环行动指南 (`行为处方`)：** 所有的向内挖掘，都是为了向外破局。通过倒逼 AI 吐出极具执行力的 To-Do / Not-To-Do 处方单，我们成功将一个单纯的“数字情绪树洞”，重构成了一台马力全开的**高阶个人成长引擎 (Pragmatic Utility)**。
