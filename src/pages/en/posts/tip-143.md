```python
def calculate_discounted_price(age, price):
    ADULT_AGE_THRESHOLD = 20
    DISCOUNT_RATE = 0.9

    if age >= ADULT_AGE_THRESHOLD:
        print("You are an adult.")
        return price * DISCOUNT_RATE

    print("You are a student.")
    return price
```

**[Improvement Points]**

1. **Function Name Change**: `calc` -> `calculate_discounted_price` (Clarifies it calculates discounted price)
2. **Magic Number Removal**: Declared numbers like `19`, `0.9` as constants to give them meaning.
3. **Unnecessary else Removal**: Used Early Return pattern to reduce indentation depth.
