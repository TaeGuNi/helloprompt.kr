- **🎯 추천 대상:** 컴활 자격증은 없지만 엑셀 복붙(Ctrl+C, V) 노가다에 지친 주니어 직장인
- **⏱️ 소요 시간:** 3시간 (수작업) → 30초 (AI 자동화 단축)
- **🤖 추천 모델:** ChatGPT (GPT-4o), Claude 3.5 Sonnet

- ⭐ **난이도:** ⭐☆☆☆☆
- ⚡️ **효과성:** ⭐⭐⭐⭐⭐
- 🚀 **활용도:** ⭐⭐⭐⭐⭐

> _"수백 개의 엑셀 파일을 하나로 합치느라 오늘 또 야근을 준비하고 계신가요? 3시간짜리 단순 반복 업무, AI 매크로를 활용하면 커피 한 잔 내리기도 전에 끝납니다."_

VLOOKUP, INDEX, MATCH, 피벗 테이블... 매번 구글에 검색하며 헤매고 계신가요?

## ⚡️ 3줄 요약 (TL;DR)

1. **상황과 목적의 구체화:** "함수 짜줘"가 아닌, 데이터의 생김새(행/열 구조)와 원하는 결과물의 형태를 정확하게 설명하세요.
2. **복잡한 작업은 VBA로:** 단순 계산을 넘어선 파일 병합, 서식 일괄 변경 등은 주저하지 말고 VBA 매크로 코드를 요청하세요.
3. **친절한 주석과 실행법 요구:** 코딩을 몰라도 직접 수정하고 실행할 수 있도록, AI에게 각 코드의 의미와 엑셀 적용 방법까지 세트로 받아내야 합니다.

## 🚀 해결책: 엑셀 자동화 마스터 프롬프트

### 🥉 Basic Version (기본형)

복잡한 조건문이나 특정 텍스트 추출 등, 당장 쓸 수식이 기억나지 않을 때 활용하세요.

> **역할:** 너는 10년 차 글로벌 IT 기업의 재무/데이터 분석 엑셀 전문가야.
>
> **요청:** 아래 데이터 구조를 보고, `[원하는 결과: 예 - B열의 이메일 주소에서 '@' 앞의 아이디만 추출]`하는 완벽한 엑셀 수식을 작성해 줘.
>
> **데이터 예시:**
>
> - A열: 사번
> - B열: 이메일 (예: hello@prompt.kr)
> - C열: 소속 부서
>
> **제약사항:** 에러(예: 이메일 주소가 비어있는 경우)가 발생하지 않도록 `IFERROR`로 감싸서 공백으로 처리해 줘.

<br>

### 🥇 Pro Version (전문가형)

매월 말일, 수십 개의 부서 취합 파일을 하나로 합치거나 특정 양식으로 변환해야 하는 '지옥의 엑셀 노가다'를 30초 만에 끝내는 프롬프트입니다.

> **역할 (Role):** 너는 마이크로소프트 MVP 출신의 엑셀 및 업무 자동화(VBA) 수석 엔지니어 역을 맡아줘. 코딩을 전혀 모르는 일반인도 쉽게 쓸 수 있는 직관적인 코드를 작성하는 것이 특징이야.
>
> **상황 (Context):**
>
> - 현재 상황: 나는 매주 `[반복 작업 내용: 예 - 50개의 팀별 영업 실적 엑셀 파일(.xlsx)을 열어서, 'Raw_Data' 시트의 내용만 하나의 마스터 파일로 합치는 작업]`을 수작업으로 하고 있어.
> - 목표: 이 끔찍한 반복 작업을 단 한 번의 클릭으로 끝낼 수 있도록 완전 자동화하고 싶어.
>
> **요청 (Task):**
>
> 1. **완벽한 VBA 매크로 작성:** 위 상황을 완벽하게 해결하는 빠르고 오류 없는 VBA 매크로 코드를 작성해 줘.
> 2. **초보자 맞춤형 가이드:** 코딩을 모르는 내가 이 코드를 엑셀 어디에, 어떻게 붙여넣고 실행해야 하는지(예: `Alt + F11` 이후의 단계) 아주 상세하고 친절하게 1단계부터 설명해 줘.
> 3. **친절한 한글 주석:** 내가 나중에 경로명이나 시트명을 직접 수정할 수 있도록, 코드의 핵심 변수 부분에 알아보기 쉬운 한글 주석을 달아줘.
> 4. **방어적 프로그래밍:** 대상 폴더에 파일이 없거나, 시트 이름이 달라서 생길 수 있는 오류를 방지하는 예외 처리(`Error Handler`) 코드를 반드시 포함해 줘.
>
> **제약사항 (Constraints):**
>
> - 데이터 구조 (Input Variables):
>   - 취합할 파일들이 있는 대상 폴더 경로: `[예: C:\Users\Desktop\이번주_영업실적\]`
>   - 복사해 올 원본 시트 이름: `[예: Raw_Data]`
>   - 데이터가 시작되는 셀 위치: `[예: A2부터 끝까지]`
>
> **주의사항 (Warning):**
>
> - 지원되지 않는 오래된 VBA 함수 사용을 지양하고 가장 호환성이 높은 최신 방식을 사용해 줘.

## 💡 작성자 코멘트 (Insight)

이 프롬프트의 진짜 가치는 **'코드 작성'을 넘어 '코드 적용 방법'까지 한 번에 얻어낸다는 점**에 있습니다. 수많은 직장인들이 AI로 코드를 짜놓고도 엑셀에 어떻게 적용하는지 몰라 결국 포기하곤 합니다.

특히 VBA를 다룰 때는 폴더 경로 설정에서 오류가 가장 많이 납니다. 만약 코드를 실행했는데 에러 팝업이 뜬다면 당황하지 말고, **"런타임 오류 '1004'가 떴어. 파일 경로를 찾을 수 없대."** 처럼 에러 메시지를 그대로 복사해서 AI에게 다시 던져보세요. 무엇을 잘못 입력했는지 족집게처럼 찾아내 수정된 코드를 제시해 줄 것입니다. 또한 Mac용 엑셀과 Windows용 엑셀은 디렉토리 경로 구조나 지원하는 VBA 함수가 미세하게 다르므로, Mac 사용자라면 프롬프트 시작 시 "나는 Mac용 엑셀 365를 쓰고 있어"라고 반드시 명시해야 합니다.

## 🙋 자주 묻는 질문 (FAQ)

- **Q: 회사 보안 정책 때문에 회사 컴퓨터에서는 ChatGPT를 못 쓰는데 어쩌죠?**
  - A: 민감한 데이터(고객 정보, 사내 기밀 등)는 절대 프롬프트에 입력하지 마세요. 대신 데이터의 **'구조(A열: 더미 이름, B열: 더미 숫자)'와 '조건'만 스마트폰이나 외부망 PC에서 AI에게 입력**하여 수식과 VBA 코드만 받아온 뒤, 회사 엑셀에 복사해서 적용하는 방식으로 안전하게 활용해야 합니다.

- **Q: AI가 짜준 코드를 실행했는데 파일 데이터가 엉망이 되면 어떡하나요?**
  - A: 매크로(VBA)는 한 번 실행하면 '실행 취소(Ctrl+Z)'가 불가능합니다! 따라서 **반드시 원본 파일을 복사하여 백업해 둔 상태에서, 복사본 파일로 매크로 코드를 테스트**하는 습관을 들여야 대참사를 막을 수 있습니다.

## 🧬 프롬프트 해부 (Why it works?)

1. **상세한 상황(Context) 부여:** 단순히 "파일 합쳐줘"라고 하지 않고 엑셀 파일의 확장자, 시트명, 복사할 범위까지 짚어주어 AI가 불필요한 가정을 하지 못하게 사전에 차단했습니다.
2. **사용자 친화적 지시(UX Focus):** 코딩 비전문가라는 점을 Role과 Task에 명확히 밝혀, AI가 복잡한 기술 용어 대신 직관적이고 친절한 한글 주석과 UI 적용 가이드를 출력하도록 강제했습니다.
3. **방어 기제(Constraints):** '예외 처리(Error Handler)' 조건을 걸어두어, 폴더가 비어있거나 파일이 손상된 현실적인 실무 에러 상황에서도 엑셀이 강제 종료되지 않도록 안전성을 확보했습니다.

## 📊 증명: Before & After

### ❌ Before (야근의 늪)

```text
1. 1팀 영업실적 파일 열기
2. 'Raw_Data' 시트 드래그해서 Ctrl+C
3. 취합용 마스터 파일에 Ctrl+V
4. 1팀 파일 닫기
5. 2팀 영업실적 파일 열기... (50번 무한 반복)
결과: 단순 복붙 노가다에만 3시간 소요. 중간에 실수로 겹쳐 쓰면 처음부터 다시 시작.
```

### ✅ After (AI 자동화의 기적)

```text
' 단축키(Alt+F8)를 누르고 "CombineSheets"를 클릭하면 끝!

Sub CombineSheets()
    Dim Path As String, Filename As String
    Dim Sheet As Worksheet

    ' [주의] 아래 경로만 본인 PC에 맞게 수정하세요!
    Path = "C:\Users\Desktop\이번주_영업실적\"
    Filename = Dir(Path & "*.xlsx")

    ' 파일이 없을 경우 친절한 안내 메시지 띄우기
    If Filename = "" Then
        MsgBox "지정된 폴더에 엑셀 파일이 없습니다. 경로를 다시 확인해주세요!", vbCritical
        Exit Sub
    End If

    ' 이하 50개의 파일을 자동으로 열고, 복사하고, 닫는 마법의 코드 실행...
    Do While Filename <> ""
        Workbooks.Open Filename:=Path & Filename, ReadOnly:=True
        ' ... (AI가 완벽하게 짜준 데이터 병합 로직) ...
        Filename = Dir()
    Loop

    MsgBox "50개 파일 취합이 완료되었습니다! 일찍 퇴근하세요! 🎉", vbInformation
End Sub
```
