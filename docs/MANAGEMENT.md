# 📅 프로젝트 관리 & 로드맵

## 🎯 목표 (Goal)

1. **콘텐츠:** 2026년까지 퀄리티 있는 프롬프트 아티클 100개 발행 (달성 완료 🎉 - 현재 298개)
2. **수익화:** 구글 애드센스 승인 및 월 방문자 1만 명(10k MAU) 달성
3. **브랜딩:** "AI 프롬프트 하면 여기!"라는 인지도 확보

- [x] **Phase 1: 구축 (Setup) - 2025 Q4**
  - Astro 프로젝트 생성 & Cyberpunk 테마 적용
  - 글 템플릿 고도화 (v4.0)
  - 배포 자동화 (`pnpm run deploy`)
- [x] **Phase 2: 기능 확장 (Feature) - 2026 Q1**
  - 실시간 검색 & 페이지네이션

  - 테스트 자동화 (TDD/E2E)
  - 다국어 지원 (i18n) & 9개국 번역
- [x] **Phase 3: 콘텐츠 확충 (Content) - 2026 Q2**

- [ ] **Phase 4: 수익화 (Monetization) - 2026 Q3**
  - 구글 서치 콘솔 등록
  - 애드센스 신청
  - KPI: MAU 10k 달성

## 📏 퀄리티 가이드라인 (Quality Rule)

1. **접근성(A11y):** 명도 대비 준수, 시맨틱 태그 사용
2. **실용성:** "복사해서 바로 쓸 수 있는가?"를 최우선으로 함
3. **가독성:** Before/After 예시 필수, 프롬프트 해부 섹션 포함
