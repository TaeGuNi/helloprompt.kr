## 📊 証明: Before & After

### ❌ Before (入力 CSV)

```csv
名前,年齢,職業
ホン・ギルドン,30,開発者
キム・チョルス,25,デザイナー
```

### ✅ After (結果 - JSON + フィールド名変更)

**リクエスト:** CSVをJSONに変えるが、英語フィールド名に変更して。

```json
[
  {
    "name": "ホン・ギルドン",
    "age": 30,
    "job": "開発者"
  },
  {
    "name": "キム・チョルス",
    "age": 25,
    "job": "デザイナー"
  }
]
```

### ✅ After (結果 - SQL Insert文)

**リクエスト:** MySQL `users`テーブルに入れられるINSERT文に変えて。

```sql
INSERT INTO users (name, age, job) VALUES
('ホン・ギルドン', 30, '開発者'),
('キム・チョルス', 25, 'デザイナー');
```
