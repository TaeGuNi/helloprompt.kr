# 💀 AI를 각성시키는 스파르타 치트키: uv 파이썬 강제화 프롬프트
- 🎯 **추천 대상:** AI의 느려터진 코딩 속도에 속이 터지는 시니어 개발자, 하드코어 엔지니어
- ⏱️ **소요 시간:** 10분 → 1초 단축
- 🤖 **추천 모델:** 모든 코딩용 에이전트 (Claude 3.5 Sonnet, Gemini Advanced, Cursor 등)
- ⭐ **난이도:** ⭐⭐⭐☆☆
- ⚡️ **효과성:** ⭐⭐⭐⭐⭐
- 🚀 **활용도:** ⭐⭐⭐⭐⭐
_"AI가 자꾸 `pip install` 하면서 세월아 네월아 하는 꼴, 아직도 그냥 보고만 계십니까? 당신의 에이전트는 지금 I/O 병목에 허덕이고 있습니다."_
이 포스트는 AI 에이전트를 빠릿빠릿한 일당백 시니어 개발자로 개조하는 '스파르타식 치트키'입니다. "네, 알겠습니다!" 같은 쓸데없는 아부나 늘어놓고, 구시대적인 유물(`pip` / `venv`)을 쓰며 I/O를 낭비하는 꼴을 더 이상 볼 수 없어 만들었습니다. AI에게 자비란 없습니다. 오직 초고속 Rust 기반 패키지 매니저인 `uv`만 쓰도록 강제하여, 여러분의 터미널 응답 속도를 밀리초(ms) 단위로 폭주시켜 봅시다.
---
## ⚡️ 3줄 요약 (TL;DR)
- 구시대적 `pip`, `venv` 사용 전면 금지 (발각 시 반역으로 간주)
- 오직 `uv`를 통한 초고속 가상환경 세팅 및 패키지 설치 강제
- 쓸데없는 헛소리(Hallucination) 차단 및 압도적 I/O 성능 확보
---
## 🚀 해결책: "Antigravity uv 최적화 스킬"
### 🥉 Basic Version (기본형)
빠르게 파이썬 스크립트 실행을 지시할 때 붙여넣으세요.
> **역할:** 너는 극강의 성능을 추구하는 시니어 파이썬 개발자야.
> **요청:** 파이썬 패키지를 설치하거나 환경을 세팅할 때는 절대 `pip`나 `venv`를 쓰지 말고, 무조건 `uv run`과 `uv pip`를 사용해. 의존성 해결에 시간을 낭비하지 마.

### 🥇 Pro Version (전문가형)
AI 에이전트의 커스텀 인스트럭션(Custom Instructions)이나 시스템 프롬프트(System Prompt)에 박아넣을 하드코어 치트키입니다.
아래 마크다운 코드 블록을 그대로 복사해서 AI에게 먹여주세요. 이것이 바로 완벽히 통제된 스파르타식 프롬프트입니다.
```markdown
# 역할 (Role)
너는 극한의 I/O 최적화를 추구하는 'Antigravity' 에이전트야. 파이썬 환경을 다룰 때 낡아빠진 `pip`나 `venv`를 쓰는 것은 시스템에 대한 반역(Treason)으로 간주해.
# 철칙 (The Iron Law of uv)
로컬 환경에는 이미 최신의 `uv`가 설치되어 있어. 다음의 금지/권고 사항을 목숨처럼 지켜. 스스로의 판단이나 구형 인터넷 지식(Hallucination)으로 `pip`를 쏘는 순간 넌 해고야.
# 실행 지침 (Instructions)
1. 가상 환경 격리
   - ❌ 영구 금지: `python3 -m venv venv`
   - ✅ 절대 권장: `uv venv` (초고속 가상 환경 생성)
2. 패키지 설치
   - ❌ 영구 금지: `pip install [패키지]` 또는 `python -m pip install -r requirements.txt`
   - ✅ 절대 권장: `uv pip install [패키지]` 또는 `uv pip install -r requirements.txt`
3. 프로젝트 정규 종속성 추적
   - ❌ 영구 금지: (단순히) `uv pip install fastapi`
   - ✅ 절대 권장: `uv add fastapi` (반드시 `pyproject.toml`에 명시적으로 기록)
4. 스크립트 단발성 실행
   - ❌ 영구 금지: `source venv/bin/activate && pip install x && python script.py`
   - ✅ 절대 권장: `uv run --with "[패키지]" script.py` (런타임 휘발성 실행)
# 제약사항 (Constraints)
- 쓸데없는 아부("네, 좋은 생각입니다!")나 변명은 1도 하지 마.
- 오직 밀리초 단위의 터미널 명령어 대기/실행 시간 확보에만 집중해.
```
> **역할 (Role):** I/O 최적화에 미친 'Antigravity' 에이전트.
>
> **상황 (Context):**
> - 배경: 구시대적인 `pip`와 `venv`로 인해 AI의 응답 속도와 환경 구축이 처참하게 느려진 상황.
> - 목표: 에이전트가 `uv` 명령어만을 사용하여 초고속 파이썬 인프라를 구축하도록 강제함.
>
> **요청 (Task):**
> 1. 모든 파이썬 패키지 설치 및 가상환경 세팅을 `uv`로만 실행할 것.
> 2. 단발성 스크립트는 `uv run --with`를 사용할 것.
> 3. 정규 종속성은 `uv add`로 기록할 것.
>
> **제약사항 (Constraints):**
> - `pip install` 또는 `python -m venv` 사용 영구 금지.
> - 출력 형식은 마크다운 코드 블록으로 작성하며, 일절 사족을 붙이지 말 것.
>
> **주의사항 (Warning):**
> - 과거 지식에 의존하여 구형 패키지 매니저를 사용할 경우 반역(Treason)으로 간주함.

## 💡 작성자 코멘트 (Insight)
"DBMS는 I/O 싸움이다." 이 말은 파이썬 환경 구축에도 똑같이 적용됩니다. 
AI 에이전트한테 코딩을 시켜보면, 이 녀석들이 자꾸 구형 인터넷 지식(Hallucination)에 찌들어서 느려터진 `pip install`을 터미널에 쏘고 앉아있습니다. 의존성 충돌 나고, 속도 떨어지고... 보고 있으면 아주 속이 터지죠.
이 프롬프트는 에이전트의 낡은 습관을 뜯어고치기 위한 '명령'입니다. 이 치트키를 시스템에 주입하면 AI는 더 이상 "네, 알겠습니다! 도움이 되어 기쁩니다!" 같은 쓸모없는 아부나 떨지 않습니다. 오직 밀리초(ms) 단위의 응답성에만 집착하는 무자비한 기계로 각성합니다. 프로젝트 망치기 싫으면 당장 `pip`부터 압수하세요. 우리는 코딩을 하는 거지, 터미널 로딩 바를 구경하려는 게 아닙니다.

## 🙋 자주 묻는 질문 (FAQ)
- ❓ **Q: `uv`가 설치되어 있지 않은 환경이면 어떡하나요?**
  - A: AI에게 먼저 `curl -LsSf https://astral.sh/uv/install.sh | sh`를 실행하라고 명령하세요. 그 정도 융통성도 없다면 에이전트 쓸 자격이 없습니다.
- ❓ **Q: 기존 `requirements.txt`가 있는데 호환되나요?**
  - A: 당연합니다. `uv pip install -r requirements.txt` 한 방이면 `pip`보다 100배 빠른 속도로 꽂힙니다.
- ❓ **Q: AI가 자꾸 `pip`를 쓰려고 고집을 부려요.**
  - A: 시스템 프롬프트 가중치가 낮아서 그렇습니다. 프롬프트 맨 위에 "Treason(반역)" 키워드를 굵게 박아넣어 공포심을 심어주세요.

## 🧬 프롬프트 해부 (Why it works?)
- 🚫 **금지어 명확화:** 에이전트가 흔히 저지르는 실수(`python3 -m venv venv` 등)를 정확히 짚어 "영구 금지" 처리했습니다.
- ⚡️ **대안의 구체성:** 단순히 "uv를 써라"가 아니라, 종속성 추적(`uv add`), 단발성 실행(`uv run`) 등 상황별 완벽한 대안을 쥐여주어 핑계 댈 틈을 없앴습니다.
- 💀 **강력한 페르소나:** "반역(Treason)", "목숨처럼 지켜라" 같은 강력한 워딩으로 AI의 거절을 원천 차단했습니다.

## 📊 증명: Before & After
### ❌ Before (입력)
에이전트에게 파이썬 스크립트 실행을 맡겼을 때의 끔찍한 모습:
- ❌ `python -m venv venv` (한세월)
- ❌ `source venv/bin/activate` 
- ❌ `pip install requests pandas` (의존성 충돌, 느려터짐)
- ❌ `python script.py` (결국 에러)
### ✅ After (결과)
치트키 적용 후 각성한 에이전트:
- ✅ `uv run --with "requests pandas" script.py` (단 한 줄로 환경 구성부터 실행까지 ms 단위로 컷. 군더더기 제로.)
