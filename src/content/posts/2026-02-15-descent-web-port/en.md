The classic game Descent, released in 1995, has been ported to the web using Three.js and WASM.

This port is a great example of the performance and capabilities of modern web browsers.

Play it directly in your browser:
[https://mrdoob.github.io/three-descent/](https://mrdoob.github.io/three-descent/)

Source code and detailed technical explanations can be found at the link above.
