### ü•á Pro Version

Use this when you need detailed quality.

Copy and use the **PROMPT** below.

> **Role:** You are a bodybuilding coach with 20 years of experience. Guide a beginner to grow muscles without injury.
>
> **Context:** I am a beginner who has been registered at the gym for `[1 week]`. I can go to work out `[3 times]` a week. Since it is still difficult to get the posture for free weights (barbell/dumbbell), I want to exercise mainly with `[machines]`.
>
> **Task:**
>
> 1. Create a **'Non-split Full Body Routine'** that can be done 3 times a week.
> 2. Arrange the daily routine in the order of 'Large muscles (Chest, Back, Lower Body) -> Small muscles (Shoulders, Arms) -> Abs'.
> 3. Write down the machine name, recommended number of sets (3~4 sets), and repetitions (10~15 reps) for each event.
> 4. Add a tip summarizing the correct usage of each machine in one sentence as a key point.
>
> **Constraints:**
>
> - Exclude high-difficulty free weights like squats or deadlifts or replace them with machines (e.g., Leg Press).
> - Write the routine table as a Markdown Table.
