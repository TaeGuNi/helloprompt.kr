# 📝 엑셀 초보 탈출: CAGR(연평균 성장률) 자동 계산기

> **🎯 추천 대상:** 누구나
> **⏱️ 소요 시간:** 5분
> **🤖 추천 모델:** 모든 AI 모델

| 난이도  |   효과성   |  활용도   |
| :-----: | :--------: | :-------: |
| ⭐⭐☆☆☆ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐☆ |

_"단순 평균의 함정에 속지 마세요. 진짜 성장은 복리로 계산해야 보입니다."_

회사의 매출 성장이나 내 주식 계좌의 수익률을 보고할 때, 단순히 `(최종 - 최초) / 최초`로 계산하면 왜곡이 생깁니다. 매년 꾸준히 성장한 비율을 나타내는 **CAGR(Compound Annual Growth Rate)**이 정답입니다. AI에게 물어보면 복잡한 수학 공식 몰라도 엑셀 수식을 바로 알려줍니다.

---

## ⚡️ 3줄 요약 (TL;DR)

1. CAGR의 개념과 단순 수익률과의 차이 이해
2. 엑셀 `RATE` 함수 또는 `POWER` 함수를 활용한 수식 생성
3. 연도별 데이터만 있으면 즉시 적용 가능한 템플릿 제공


### ✅ After (결과)

````text
### 📈 CAGR(연평균 성장률) 엑셀 수식

**1. 기본 원리**
- (도착 값 / 출발 값) ^ (1 / 기간) - 1

**2. 엑셀 수식 (복사해서 쓰세요!)**

**옵션 A: 수식 직접 입력 (POWER 함수)**
```text
=POWER(B2/B1, 1/(A2-A1)) - 1
```
````

**옵션 B: 재무 함수 활용 (RATE 함수)**

```text
=RATE(A2-A1, 0, -B1, B2)
```

_(주의: 시작 값인 B1 앞에 마이너스(-)를 붙여야 오류가 안 납니다!)_

**3. 서식 꿀팁**

- 결과값이 `0.2011`로 나온다면? 단축키 **[Ctrl + Shift + %]**를 누르세요. `20.1%`로 예쁘게 바뀝니다.

