## 🙋 Domande Frequenti (FAQ)

- **Q: È possibile utilizzare questo prompt anche in ecosistemi moderni come Tailwind CSS o Styled-Components?**
  - A: Assolutamente sì! Ti basterà aggiungere una singola riga di istruzione nel Contesto o nella Richiesta: *"Scrivi il codice sfruttando le classi di estensione `animate-` di Tailwind e fornisci l'esatta configurazione da inserire in `tailwind.config.js`"*, oppure *"Scrivilo utilizzando l'helper `keyframes` tipico di Styled-Components"*. L'AI mapperà il codice adattandolo chirurgicamente al tuo stack tecnologico.

- **Q: L'AI è in grado di convertire le istruzioni in sintassi specifiche come Framer Motion per React o le Transition di Vue?**
  - A: Certamente. Specificando la tua libreria di riferimento, ad esempio chiedendo *"Scrivi un componente animato sfruttando le proprietà `initial`, `animate` e `transition` native di Framer Motion in React"*, l'intelligenza artificiale genererà un codice perfettamente ottimizzato per esaltare i punti di forza di quella specifica libreria.

- **Q: Ho applicato il codice fornito dall'AI, ma sul browser l'animazione scatta vistosamente. Come posso risolvere?**
  - A: Nel 99% dei casi, questi crolli di framerate (Jank) sono causati dal tentativo di animare proprietà strutturali che scatenano un letale ricalcolo del layout (Reflow), come `width`, `height`, `top` o `left`. Come esplicitamente sottolineato nei vincoli del nostro prompt Pro, devi sempre ordinare all'AI di manipolare **esclusivamente** `transform: translate()` e `opacity`.

## 📊 La Prova: Prima & Dopo (Before & After)

### ❌ Prima (Metodo manuale tradizionale)

```css
/* Modifica manuale dei valori di layout basandosi sull'istinto e infiniti refresh della pagina... */
/* Il Reflow causa un'animazione a scatti e dall'aspetto economico */
.toast {
  animation: slide-in 3s ease;
}

@keyframes slide-in {
  0% {
    top: -50px;
    opacity: 0;
  }
  100% {
    top: 20px;
    opacity: 1;
  }
}
```

### ✅ Dopo (Applicando il prompt AI)

```css
/* Ottimizzazione hardware tramite GPU e tensione perfetta con un Cubic-bezier custom! */
.toast-jelly {
  animation: jelly-slide-fade 3s cubic-bezier(0.34, 1.56, 0.64, 1) forwards;
  will-change: transform, opacity;
}

@keyframes jelly-slide-fade {
  0% {
    transform: translateY(-100px) scale(0.8);
    opacity: 0;
  }
  40% {
    transform: translateY(10px) scale(1.05);
    opacity: 1;
  }
  60% {
    transform: translateY(0) scale(1);
    opacity: 1;
  }
  80% {
    transform: translateY(0) scale(1);
    opacity: 1;
  }
  100% {
    transform: translateY(-50px) scale(0.9);
    opacity: 0;
  }
}
```
