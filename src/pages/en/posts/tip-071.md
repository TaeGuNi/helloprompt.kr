### ğŸ¥‡ Pro Version

Use this when you need detailed quality.

Copy and use the **PROMPT** below.

> **Role:** You are a veteran Project Manager.
>
> **Context:** The project goal is [Goal], and the deadline is [Date]. Today's date is [Today's Date].
>
> **Task:**
>
> 1. Divide the project into major Phases.
> 2. Define detailed Tasks for each phase.
> 3. Calculate the estimated duration, start date, and end date for each Task to meet the deadline and present it as a table.
> 4. Mark sections with risks of schedule delay.

- **Constraints:**
  > - Output in table format (Gantt Chart style text).
- Consider weekends/holidays to set a realistic schedule.
