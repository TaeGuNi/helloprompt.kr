_"会社のセキュリティポリシーのせいでChatGPTにコードを貼れない？ かといって古いモデルは使いたくない？ ならば、**DeepSeek R1**をあなたのPCに閉じ込めて、思う存分こき使いましょう。100%無料、100%プライベートです。"_

2026年現在、開発者の間で最もホットなキーワードは間違いなく **"Local LLM"** です。中でも **DeepSeek R1** は「オープンソース界のミス」と呼ばれるほど異常な性能（特にコーディング能力）を見せています。この記事では、あなたのMac（またはWindows PC）にDeepSeekを最も簡単かつ迅速にインストールする方法を紹介します。

---

## ⚡️ 3行要約 (TL;DR)

1.  **Ollama** をインストール (LLM実行ランナー)
2.  ターミナルでコマンドを1行入力 (`ollama run deepseek-r1`)
3.  VS Codeやブラウザで無制限の無料コーディング

---

## 🚀 Step 1: Ollamaのインストール

まず、LLMを動かすエンジンである **Ollama** をインストールします。Dockerよりもはるかに簡単です。

### Mac / Linux

ターミナルを開き、以下のコマンドをコピー＆ペーストしてください。

```bash
curl -fsSL https://ollama.com/install.sh | sh
```

### Windows

[Ollama公式サイト](https://ollama.com)から `Download for Windows` をクリックしてインストーラーを実行してください。

---

## 🚀 Step 2: DeepSeek R1 召喚

次に、モデルをダウンロードして実行します。PCのスペックに合わせて選んでください。

### 🥉 Basic: 8Bモデル (一般ノートPC用)

M1/M2 MacBook Air、一般的なゲーミングノートPCなどでサクサク動きます。

```bash
ollama run deepseek-r1:8b
```

### 🥇 Pro: 70Bモデル (ハイスペックPC/M3 Max用)

メモリ32GB以上推奨。GPT-4クラスの推論能力を発揮します。

```bash
ollama run deepseek-r1:70b
```

コマンドを打つと自動的にダウンロードが始まり、完了するとすぐにチャットプロンプトが表示されます。

---

## 🚀 Step 3: 実践活用 (VS Code連携)

ターミナルでチャットするのは味気ないですよね。**VS Code** に連携させて、真のCopilotにしましょう。

1.  VS Codeの拡張機能タブで **"Continue"** を検索してインストール。
2.  インストール後、左側のサイドバーアイコンをクリック → `Add Model` を選択。
3.  Providerで `Ollama` を選択。
4.  Modelで `DeepSeek R1` を選択。(Autodetectを押せば自動認識します)
5.  これでコードを選択して `Cmd + L` (Windowsは `Ctrl + L`) を押せばすぐに質問できます。

**Q:** "この関数をリファクタリングして。"
**A:** (DeepSeekが瞬時にコードを吐き出します)

## 🧬 プロンプト解剖 (Why it works?)

「あれ、これインストールガイドじゃないの？ プロンプトは？」
お待たせしました。**DeepSeekに仕事をさせる時に最も効果的なプロンプト**です。DeepSeekは「思考プロセス(Chain of Thought)」を重視するため、**「考えてみて」**と指示するのが鍵です。

### DeepSeek専用コーディングプロンプト

**役割 (Role):** 君はGoogleのシニアエンジニアであり、クリーンコードの伝道師だ。

**状況 (Context):** 私が書いたこのコードは動くけど、可読性が悪く非効率だ。

**依頼 (Task):**

1. コードの問題点（時間計算量、命名など）を先に分析して。( **思考プロセス(Thinking Process)を先に出力すること** )
2. 最適化されたコードを書いて。
3. なぜそう直したのか、コメントで説明して。
