# 📝 難しいお金の話、丁寧だが断固とした「年俸引き上げ要請メール」ライター

**🎯 おすすめ対象:** 会社員
**⏱️ 所要時間:** 5分
**🤖 推奨モデル:** 全てのAIモデル

| 難易度  |    効果    |   実用性   |
| :-----: | :--------: | :--------: |
| ⭐⭐☆☆☆ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |

_「会って話したいけど、どうやってスケジュールを掴めばいいのか…？」_

対面交渉の前、**「ドアを開けるメール」**が重要です。いきなり「お金の話をしましょう」と言えば、上司は防御的になります。「成果」と「期待」をほのめかしながら、丁寧にミーティングを提案する技術が必要です。

---

## ⚡️ 3行要約 (TL;DR) {#tl-dr}

1. 「成果レビューおよび報酬調整」のためのミーティング日程を掴む目的で作成する。
2. 一年間の主な成果を短く要約し、上司に想起させる。
3. 丁寧だが目的を明確にするプロフェッショナルなメールを完成させる。

---

## 🚀 解決策: 「年俸交渉メールライター」

### 🥉 基本バージョン

手っ取り早く結果が必要な場合に使用してください。

**役割:** あなたはビジネスコミュニケーションの専門家です。
**リクエスト:** 気分を害さず丁寧に、しかし成果の根拠と希望範囲を明確に明らかにし、年俸交渉を要請するビジネスメールを作成してください。

<br>

### 🥇 プロバージョン (エキスパート)

詳細なクオリティが必要な場合に使用してください。

以下の**PROMPT**の内容をコピーして使用してください。

**役割:** あなたはビジネスコミュニケーションの専門家です。

**コンテキスト:** 私は上司の`[上司名/役職]`に年俸交渉ミーティングを要請するメールを送りたいです。

**キーポイント:**

1. トピック: 今年の成果レビューおよび来年の年俸調整に関する議論。
2. 私の成果: `[核心成果: 例 - コスト20%削減]`
3. トーン: 丁寧で肯定的だが、「交渉」という目的を明確にする。

**タスク:**

1. プロフェッショナルで明確な件名（Subject Line）を作成してください。
2. 本文で成果に自然に言及し、ミーティング日程を提案してください。
3. 具体的な金額はメールで言及せず、対面で話すように誘導してください。

**制約事項:**
1分以内に読めるように簡潔に作成してください。


### ✅ After (プロフェッショナル)

```text
件名: 2025年成果レビューおよび処遇に関するミーティング要請
チーム長様、
... 今年の「クラウド移管プロジェクト」を通じて、20%のコスト削減を達成しました。
このような結果に基づき、来年の方向性と処遇について議論する時間を持ちたいと思います。
よろしければ...
```

## 🎯 結論

よく書かれたメール一通が、ミーティング開始前から**主導権（Control）**を握ります。

スマートに交渉のドアを叩いてください。 🚪
