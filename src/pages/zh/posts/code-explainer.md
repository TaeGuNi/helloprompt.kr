## 📊 证明: Before & After

### ❌ Before (输入 - Python) {#python}

```python
result = [x * 2 for x in range(10) if x % 2 == 0]
```

### ✅ After (提示词结果)

**[摘要]**
从 0 到 9 的数字中只选出偶数，把那个值变成 2 倍后保存到新列表的代码。

**[逐行分析]**

```python
# 从 0 到 9 (range(10)) 逐个取出数字放入变量 x。 {#range-10}
# 如果 (if) x 除以 2 的余数是 0 (是偶数的话)， {#if}
# 把 x 乘以 2 的值放入列表。
result = [x * 2 for x in range(10) if x % 2 == 0]
```

**[语法要点: 列表推导式]**
Python 特有的语法，是用一行简洁地使用 `for` 循环和 `if` 条件句生成列表时使用的。

- 结构: `[表达式 for 项目 in 可迭代对象 if 条件]`
