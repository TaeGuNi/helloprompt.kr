## 📊 证明: Before & After

### ❌ Before (输入代码 - 难懂的 C 语言风格)

```c
int proc(int v) {
  int r = 1;
  while (v > 1) {
    r *= v--;
  }
  return r;
}
```

### ✅ After (解读结果)

**📝 摘要:**
这段代码是计算数学的**阶乘 (Factorial)** 的函数。返回从输入的数字开始到 1 为止所有数字相乘的值。

**🔍 详细逻辑:**

1. 将存储结果的变量 `r` 设为 1。
2. 持续重复直到输入值 `v` 大于 1。
   - 将当前 `v` 值乘到 `r` 上。
   - 将 `v` 减少 1。
3. 返回最终计算的 `r`。

**⚠️ 注意事项:**
如果输入值 `v` 太大，结果值会超出整数型 (int) 范围，发生**溢出 (Overflow)**，可能会出现莫名其妙的负数值。处理大数时必须更改类型。
