## 📊 証明: Before & After

### ❌ Before (入力コード - 難解なC言語スタイル)

```c
int proc(int v) {
  int r = 1;
  while (v > 1) {
    r *= v--;
  }
  return r;
}
```

### ✅ After (解釈結果)

**📝 要約:**
このコードは数学の**階乗(Factorial)**を計算する関数です。入力された数字から1まですべての数字を掛けた値を返します。

**🔍 詳細ロジック:**

1. 結果を保存する変数`r`を1に設定します。
2. 入力値`v`が1より大きい間ずっと繰り返します。
   - 現在の`v`値を`r`に掛けます。
   - `v`を1だけ減らします。
3. 最終計算された`r`を返します。

**⚠️ 注意事項:**
入力値`v`が大きすぎると結果値が整数型(int)範囲を超えて**オーバーフロー(Overflow)**が発生しとんでもない負の値が出ることがあります。大きな数を扱う時はタイプを変更しなければなりません。
