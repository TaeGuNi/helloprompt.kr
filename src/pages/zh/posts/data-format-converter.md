## 📊 证明: Before & After

### ❌ Before (输入 CSV) {#csv}

```csv
名字,年龄,职业
洪吉童,30,开发者
金哲秀,25,设计师
```

### ✅ After (结果 - JSON + 字段名变更) {#json}

**请求:** 把 CSV 换成 JSON，但变更为英语字段名。

```json
[
  {
    "name": "洪吉童",
    "age": 30,
    "job": "开发者"
  },
  {
    "name": "金哲秀",
    "age": 25,
    "job": "设计师"
  }
]
```

### ✅ After (结果 - SQL Insert 语句) {#sql-insert}

**请求:** 换成能放入 MySQL `users` 表的 INSERT 语句。

```sql
INSERT INTO users (name, age, job) VALUES
('洪吉童', 30, '开发者'),
('金哲秀', 25, '设计师');
```
