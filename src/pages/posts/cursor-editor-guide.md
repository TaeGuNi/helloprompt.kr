# 🖱️ Cursor 에디터 200% 활용법: 아직도 VS Code 쓰세요?

- **🎯 추천 대상:** Copilot보다 더 똑똑한 AI를 원하는 개발자, 남의 코드 분석하느라 야근하는 신입
- **⏱️ 소요 시간:** 5분 (설치 및 적응)
- **🤖 추천 모델:** Cursor (Claude 3.5 Sonnet 내장)

- ⭐ **난이도:** ⭐☆☆☆☆
- ⚡️ **효과성:** ⭐⭐⭐⭐⭐
- 🚀 **활용도:** ⭐⭐⭐⭐⭐

_"VS Code에 Copilot 붙여 쓰면 되는 거 아니야?"_

아닙니다. Cursor는 **"에디터 자체가 AI"**입니다. 단순히 다음 줄을 추천하는 게 아니라, **프로젝트 전체 파일**을 읽고 문맥을 파악해서 코드를 짜줍니다. "이 함수 고치면 저 파일에서 에러 날 텐데?"까지 알려주는 괴물이죠.

---

## ⚡️ 3줄 요약 (TL;DR) {#tl-dr}

1.  `Cmd + K`: 커서 위치에서 바로 AI에게 코드 생성을 시킨다.
2.  `Cmd + L`: 챗봇을 열고 "이 프로젝트 전체 구조 설명해줘"라고 묻는다.
3.  `@Codebase`: 채팅할 때 `@Codebase`를 태그하면 전체 파일을 참조해서 답변한다.

---

## 🚀 해결책: "Cursor Master Prompt"

### 🥉 Basic Version (단축키 마스터)

가장 많이 쓰는 기능.

> **단축키:** `Cmd + K` (Generate)
> **프롬프트:** "이 함수를 비동기(`async/await`)로 바꾸고, 에러 처리를 추가해줘."

<br>

### 🥇 Pro Version (코드베이스 전체 리팩토링)

파일 하나가 아니라, 프로젝트 전체에 영향을 미치는 수정을 할 때.

> **단축키:** `Cmd + L` (Chat) -> `@Codebase` 태그 필수
>
> **요청 (Prompt):**
> "현재 프로젝트의 `auth` 관련 로직을 전부 찾아서, JWT 토큰 만료 시간을 1시간에서 24시간으로 늘려줘.
> 그리고 `login` 함수에서 토큰 갱신(Refresh) 로직이 누락된 것 같은데, 확인하고 추가해줘.
> 관련된 모든 파일(`utils`, `api`, `store`)을 수정하고, 수정된 파일 목록을 알려줘."

## 💡 작성자 코멘트 (Insight) {#insight}

Cursor는 **"개발자의 뇌를 확장하는 도구"**입니다.
단순히 코드를 대신 짜주는 것을 넘어,
"이 코드는 왜 이렇게 짰지?"라고 물어보며(Explain) **코드 리뷰어**로 활용할 때 진가를 발휘합니다.
특히 오픈소스 분석할 때 `@Codebase` 태그는 사기급 기능입니다.

## 🧬 기능 해부 (Why it works?) {#why-it-works}

1.  **@Codebase:** 이게 핵심입니다. 기존 Copilot은 현재 열려있는 파일만 보지만, Cursor는 `@Codebase`를 통해 **전체 프로젝트를 인덱싱**합니다. "A 파일 고치면 B 파일도 고쳐야 해"라는 걸 압니다.
2.  **Diff View:** AI가 제안한 코드를 바로 적용하지 않고, `Diff` 화면으로 보여주며 **"이거 맞아? 적용할까?"** 하고 물어봅니다. (`Cmd + Y`로 수락)
