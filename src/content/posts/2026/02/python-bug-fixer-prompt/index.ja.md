## 📊 証明: Before & After

### ❌ Before (入力)

```python
def calculate_average(numbers):
    total = sum(numbers)
    return total / len(numbers)

print(calculate_average([]))
```

エラー: `ZeroDivisionError: division by zero`

### ✅ After (結果)

**エラーの原因:**
空のリスト `[]` を引数として渡したため、`len(numbers)` が `0` となり、ゼロ除算（ZeroDivisionError）が発生しました。

**修正後のコード:**

```python
def calculate_average(numbers):
    if not numbers:  # リストが空の場合は0を返すか、適切に処理します
        return 0
    total = sum(numbers)
    return total / len(numbers)

print(calculate_average([]))  # 出力: 0
```

**解説:**
Pythonではリストが空かどうかを判定する際、`if not numbers:` とするアプローチが最もPythonicです。ゼロ除算を未然に防ぐため、リストの要素数を確認するロジックを追加しました。
