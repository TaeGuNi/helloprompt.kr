# (Updated) 📝 브라우저 속의 스파이, 확장프로그램 {#updated}

- **🤖 관련 기술:** Chrome Extensions, Spyware, Data Privacy

> _"편리하려고 깐 확장프로그램이, 내 모든 방문 기록을 팔아넘기고 있다면?"_

크롬 웹 스토어에 등록된 약 280개의 확장프로그램이 사용자의 브라우징 데이터를 몰래 수집하여 제3자에게 판매하거나 스파이웨어로 동작한다는 충격적인 보고서가 나왔습니다. 영향을 받은 사용자만 수백만 명에 달합니다.

## (Updated) ⚡️ 3줄 요약 (TL;DR) {#tl-dr}

1. **현황**: 280개 이상의 악성 확장프로그램 적발 (PDF 변환, VPN, 날씨 앱 등).
2. **수법**: 정상적인 기능을 제공하는 척하면서 백그라운드에서 URL, 쿠키, 검색어 수집.
3. **대책**: 불필요한 확장프로그램 삭제 및 권한 설정 확인 ("모든 사이트에서 데이터 읽기" 권한 주의).
