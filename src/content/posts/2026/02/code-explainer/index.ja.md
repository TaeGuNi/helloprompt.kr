## 📝 このコード、どういう意味ですか？親切なコード解説士

- **🎯 おすすめ対象:** ジュニア開発者、プログラミング学習者、非エンジニアのIT担当者
- **⏱️ 所要時間:** 30分 → 1分に短縮
- **🤖 推奨モデル:** すべての対話型AI (ChatGPT, Claude, Geminiなど)

- ⭐ **難易度:** ⭐☆☆☆☆
- ⚡️ **効果性:** ⭐⭐⭐⭐⭐
- 🚀 **活用度:** ⭐⭐⭐⭐⭐

> _「動くことは動くけれど… なぜ動いているのか全く分からない…」_

開発者にとって永遠のミステリー。Stack Overflowからコピーしたコードが奇跡的に動いたものの、自分のプロジェクトに合わせてどう修正すべきか途方に暮れた経験はありませんか？あるいは、前任者が残した暗号のようなレガシーコードを解読する羽目になり、頭を抱えているかもしれません。

このプロンプトは、そんなあなたの**専属AIメンター**となり、どんなに難解なコードでも優しく丁寧に紐解いてくれます。

---

## ⚡️ 3行まとめ (TL;DR)

1. コード全体のロジックと本来の目的を、誰にでも分かる言葉で要約します。
2. 難解なブロックに**注釈（コメント）**を付与し、行単位で徹底的に分析します。
3. 初心者が見落としがちな特有の言語構文や関数についても、追加で丁寧に解説します。

---

## 🚀 解決策：「コード難読化解除機」プロンプト

### 🥉 Basic Version (基本形)

手っ取り早く結果だけを知りたい場合に使用してください。

> **役割:** あなたは`[親切なシニアエンジニア]`です。
> **タスク:** 以下の`[コード]`がどのような処理を行っているのか、一行ずつ注釈（コメント）を追加し、初心者にも分かりやすく解説してください。

### 🥇 Pro Version (専門家形)

より深く、根本的な理解を求める場合はこちらを使用してください。

> **役割 (Role):** あなたは`[親切なシニアエンジニアであり、私の専属メンター]`です。
>
> **状況 (Context):**
>
> - 背景: 私はこのコードが正確に何をするのか、どのような原理で動いているのかを理解したいと考えています。
> - 目標: 私は`[初級/中級]`レベルのエンジニアです。私の目線に合わせて、分かりやすく解説してください。
>
> **タスク (Task):**
>
> 1. **[要約]**: このコードのコアとなる機能と処理の流れを3つの文で要約してください。
> 2. **[行単位の分析]**: コードのすべての行（または重要なブロック）に**丁寧な注釈（コメント）**を追加して書き直してください。
> 3. **[文法ポイント]**: ここで使われている特有の文法やライブラリの関数があれば、別途詳しく説明してください（例：リスト内包表記、アロー関数、正規表現など）。
>
> **制約事項 (Constraints):**
>
> - 難解な専門用語の羅列は避け、例え話などを交えて直感的な表現を使用してください。
> - 各変数がどのような役割を担っているのかを明確に指摘してください。
>
> **入力コード:**
>
> `[ここにコードを貼り付けてください]`

実際に私自身が実務で複雑なレガシーシステムを引き継いだ際、このプロンプトを駆使して数千行に及ぶコードを少しずつ解読した結果、業務の引き継ぎにかかる時間を半分以下にまで劇的に短縮することに成功しました。

## 🧬 プロンプト解剖 (Why it works?)

1. **専属メンターとしてのペルソナ付与:** AIに対して「シニアエンジニア」という明確な役割を与えることで、無味乾燥な機械的翻訳ではなく、後輩を優しく指導するような、温かみのある分かりやすいトーンを強制的に引き出しています。
2. **学習者レベルの明確な提示:** `[初級/中級]`というように現在のスキルレベルを明示することで、AIが「どこまでの前提知識を省略してよいか」を正確に判断し、難しすぎず易しすぎない最適な塩梅の解説を提供します。
3. **出力を構造化し認知負荷を軽減:** 「要約」「行単位の分析」「文法ポイント」という3段階に出力を分割することで、全体像の俯瞰から細部の深掘りへと至る、人間の自然な認知プロセスに沿った効果的な学習フローを実現しています。

## 📊 証明: Before & After

### ❌ Before (入力 - Python)

```python
result = [x * 2 for x in range(10) if x % 2 == 0]
```

### ✅ After (プロンプト適用結果)

**[要約]**

1. 0から9までの数値の中から、偶数のみを正確に抽出します。
2. 抽出された各偶数の値をそれぞれ2倍に計算します。
3. 最終的にその計算結果を格納した新しいリストを生成するコードです。

**[行単位の分析]**

```python
# range(10)によって、0から9までの数値を順番に変数xへと代入します。
# if x % 2 == 0: xを2で割った余りが0（すなわち偶数）であるかどうかを判定し、
# x * 2: 条件を満たすxを2倍して、新しいリストの要素として順次追加していきます。
result = [x * 2 for x in range(10) if x % 2 == 0]
```

**[文法ポイント]**

- **リスト内包表記 (List Comprehension):** Python特有の極めて強力な文法です。通常であれば数行にわたって記述しなければならない `for` ループや `if` 条件文を、わずか1行で簡潔かつ高速に処理できるエレガントな記述方法です。
- **基本構造:** `[式 for 変数 in イテラブルオブジェクト if 条件]`
