## 🚀 La Solution : « Maître du Codage VBA »

### 🥉 Version Basique (Basic Version)

Utilisez cette version lorsque vous avez simplement besoin d'un résultat rapide pour une tâche basique.

> **Rôle :** Tu es un Développeur Expert en Excel VBA.
> **Requête :** Je souhaite automatiser la tâche suivante sur Excel : `[Décrivez votre tâche, ex. : Supprimer toutes les lignes vides et mettre en gras les cellules contenant "Total"]`. Rédige le code VBA complet et explique-moi étape par étape comment l'intégrer, car je suis totalement débutant.

### 🥇 Version Pro (Expert Version)

Privilégiez cette version pour les processus complexes exigeant une stabilité de niveau professionnel et des performances optimales.

> **Rôle (Role) :** Tu es un Développeur Expert en Excel VBA possédant plus de 10 ans d'expérience en entreprise.
>
> **Contexte (Context) :**
>
> - Objectif : Je souhaite automatiser `[Insérez la description détaillée de la tâche répétitive, ex. : la consolidation de 5 fichiers de ventes mensuelles présents dans le même dossier]` dans Excel.
> - Niveau de l'utilisateur : Je n'ai absolument aucune notion de codage.
>
> **Requête (Task) :**
>
> 1. Rédige un code de macro VBA optimisé et sans la moindre erreur pour accomplir cette tâche.
> 2. Ajoute des commentaires explicatifs détaillés (en français) sur chaque ligne ou bloc de code afin que je comprenne parfaitement son fonctionnement.
> 3. Rédige un guide étape par étape ultra-clair et infaillible (ex. : appuyer sur Alt+F11, faire un clic droit, insérer un module, etc.) pour m'expliquer comment implémenter, sauvegarder et exécuter ce code dans mon fichier Excel.
>
> **Contraintes (Constraints) :**
>
> - Inclus obligatoirement une gestion des erreurs (`On Error GoTo`) afin d'éviter les plantages brutaux et d'afficher une boîte de dialogue claire en cas de problème.
> - Intègre des commandes d'optimisation des performances au début et à la fin du code (ex. : `Application.ScreenUpdating = False`, `Application.Calculation = xlCalculationManual`).
>
> **Avertissement (Warning) :**
>
> - Si ma demande manque de clarté concernant la structure de mes données (noms de colonnes, feuilles), pose-moi des questions de clarification avant de générer le moindre code. Ne devine absolument rien.

## 🙋 Foire Aux Questions (FAQ)

- **Q : Ces macros fonctionnent-elles sur la version Excel pour Mac ?**
  - R : Oui, le langage VBA est globalement compatible. Toutefois, les raccourcis clavier (comme Alt+F11) et l'interface de l'éditeur diffèrent légèrement sous macOS. Précisez simplement _"Je suis sur Mac"_ dans le contexte de votre prompt afin que l'IA adapte les instructions de son tutoriel.

- **Q : L'IA est-elle capable de comprendre la structure exacte de mon tableau ?**
  - R : Absolument ! Si vous utilisez ChatGPT Plus ou Claude 3, la meilleure approche consiste à exporter un petit extrait de vos données au format CSV, à le joindre dans la conversation, et à demander à l'IA d'adapter le code _exactement_ aux noms de vos colonnes. Cela permet de réduire les erreurs à zéro.

- **Q : Mon entreprise bloque les macros pour des raisons de sécurité, que puis-je faire ?**
  - R : C'est une situation courante. Demandez simplement à l'IA de coder la même solution, mais en utilisant **Office Scripts (TypeScript)** au lieu de VBA. Office Scripts est l'alternative moderne et sécurisée aux macros pour Excel sur le Web et Microsoft 365, et l'IA maîtrise parfaitement sa génération.

## 📊 Preuve : Avant & Après

### ❌ Avant (Le cauchemar manuel)

```text
(Temps perdu : 45 minutes chaque lundi matin)
Ouvrir les 12 fichiers régionaux un par un.
Copier la plage A2:H500.
Aller dans le Master.xlsx, trouver la dernière ligne vide.
Coller les valeurs seules.
Supprimer toutes les lignes où la colonne 'Statut' est vide.
Mettre en rouge les montants négatifs.
...et répéter sans faire d'erreur.
```

### ✅ Après (Résultat en 1 clic)

```vb
Sub ConsoliderEtNettoyerRapports()
    ' Optimisation extrême des performances
    Application.ScreenUpdating = False
    Application.Calculation = xlCalculationManual
    Application.EnableEvents = False

    On Error GoTo ErrorHandler

    ' [Le code généré par l'IA effectuant la consolidation de 12 fichiers en 2 secondes]

    ' Restauration des paramètres par défaut
SortiePropre:
    Application.ScreenUpdating = True
    Application.Calculation = xlCalculationAutomatic
    Application.EnableEvents = True
    MsgBox "Consolidation et nettoyage terminés avec succès en un temps record !", vbInformation
    Exit Sub

ErrorHandler:
    MsgBox "Une erreur inattendue est survenue : " & Err.Description, vbCritical
    Resume SortiePropre
End Sub
```
