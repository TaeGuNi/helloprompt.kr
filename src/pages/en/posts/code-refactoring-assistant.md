# 📝 내 코드가 스파게티? 리팩토링 전문가에게 맡겨보세요

- **🎯 추천 대상:** 주니어 개발자, 코드 리뷰어
- **⏱️ 소요 시간:** 1시간 → 10분 단축
- **🤖 추천 모델:** ChatGPT (GPT-4), Claude 3 Opus (코드 품질 우수)

- ⭐ **난이도:** ⭐⭐☆☆☆
- ⚡️ **효과성:** ⭐⭐⭐⭐⭐
- 🚀 **활용도:** ⭐⭐⭐⭐⭐

> _"돌아가는 쓰레기(Working Garbage)를 만들지 마세요. 읽기 좋은 코드가 좋은 코드입니다."_

기능은 동작하지만 유지보수가 어려운 코드는 결국 기술 부채가 됩니다. 이 프롬프트는 복잡한 로직을 단순화하고, 변수명을 직관적으로 수정하며, 최신 문법을 적용하여 코드 품질을 높여줍니다.

---

## ⚡️ 3줄 요약 (TL;DR)

1. 변수명/함수명 개선 및 중복 코드 제거
2. 최신 문법(ES6+ 등) 적용
3. 주석 및 문서화 자동 생성

---

## 🚀 해결책: "Clean Code Mentor"

### 🥉 Basic Version (기본형)

빠르게 코드 개선이 필요할 때 사용하세요.

> **역할:** 너는 클린 코드 전도사야.
> **요청:** 이 코드를 더 읽기 좋게(가독성) 리팩토링해줘.
> **코드:** `[코드 블록]`

<br>

### 🥇 Pro Version (전문가형)

코드 리뷰 수준의 피드백이 필요할 때 사용하세요.

> **역할 (Role):** 너는 10년 차 시니어 소프트웨어 엔지니어(Senior Software Engineer)야.
>
> **상황 (Context):**
>
> - 언어: `[프로그래밍 언어]`
> - 목적: `[성능 최적화 / 가독성 향상 / 버그 수정]`
>
> **요청 (Task):**
>
> 1. 제공된 코드를 분석하여 문제점(Code Smell)을 나열해줘.
> 2. 리팩토링된 코드를 작성해줘. (최신 문법 활용)
> 3. 변경된 부분에 대한 설명을 주석(Comment)으로 달아줘.
> 4. (선택) 단위 테스트(Unit Test) 코드도 작성해줘.
>
> **제약사항 (Constraints):**
>
> - 변수명은 의미가 명확하게(Descriptive) 바꿔줘.
> - 함수는 하나의 기능만 하도록(Single Responsibility Principle) 분리해줘.
> - 에러 처리를 꼼꼼하게 해줘.
>
> **주의사항 (Warning):**
>
> - 기존 로직의 동작이 변경되면 안 돼.

## 💡 작성자 코멘트 (Insight)

리팩토링 후에는 반드시 테스트 코드를 돌려 기능이 정상 동작하는지 확인하세요. AI가 제안한 코드를 무조건 신뢰하기보다, 왜 이렇게 변경했는지 이유를 물어보며 학습하는 것이 성장에 큰 도움이 됩니다.

## 📊 증명: Before & After

### ❌ Before (입력)

```javascript
function a(n) {
  if (n < 2) return n;
  return a(n - 1) + a(n - 2);
}
```

### ✅ After (결과)

```javascript
/**
 * 피보나치 수열의 n번째 값을 계산합니다. (재귀 방식)
 * @param {number} n - 계산할 순번 (0 이상의 정수)
 * @returns {number} 피보나치 수
 */
function getFibonacci(n) {
  if (n < 0) throw new Error("입력값은 0 이상이어야 합니다.");
  if (n < 2) return n;
  
  return getFibonacci(n - 1) + getFibonacci(n - 2);
}
```
