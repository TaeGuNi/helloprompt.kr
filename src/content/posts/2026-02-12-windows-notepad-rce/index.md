# (Updated) (Updated) (Updated) 📝 가장 안전할 줄 알았던 '메모장'의 배신 {#updated}

- **🎯 추천 대상:** 윈도우 사용자, 시스템 관리자
- **⏱️ 읽는 시간:** 2분
- **🤖 관련 기술:** Windows Notepad, RCE, Security Patch

_"텍스트 파일 하나 열었을 뿐인데 해킹당할 수 있다니, 믿기지 않으시죠?"_

윈도우의 기본 텍스트 에디터인 **메모장(Notepad)**에서 심각한 보안 취약점(**CVE-2026-20841**)이 발견되었습니다. 특수하게 조작된 텍스트 파일을 메모장으로 열 경우, 공격자가 원격으로 코드를 실행(RCE)할 수 있는 문제입니다.

---

## ⚡️ 3줄 요약 (TL;DR) {#tl-dr}

1. **취약점**: 윈도우 메모장 앱의 파싱 오류를 이용한 원격 코드 실행.
2. **위험성**: 이메일 첨부파일이나 다운로드한 텍스트 파일을 여는 것만으로 감염 가능.
3. **대응**: 윈도우 업데이트(Microsoft Store 앱 업데이트)를 통해 즉시 패치 필요.

---

## 💡 작성자 코멘트 (Insight) {#insight}

"메모장은 단순해서 안전하다"는 통념이 깨졌습니다. 레거시 코드와 새로운 기능(탭 기능 등)이 섞이면서 예상치 못한 버그가 발생한 것으로 보입니다. **출처가 불분명한 .txt 파일도 이제는 의심해야 하는 시대**입니다. 즉시 업데이트하세요.
