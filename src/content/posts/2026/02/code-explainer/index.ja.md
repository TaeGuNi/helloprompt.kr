# 📝 このコード、どういう意味ですか？親切なコード解説士

- **🎯 おすすめ対象:** ジュニア開発者、プログラミング学習者、非エンジニアのIT担当者
- **⏱️ 所要時間:** 30分 → 1分に短縮
- **🤖 推奨モデル:** すべての対話型AI (ChatGPT, Claude, Geminiなど)

- ⭐ **難易度:** ⭐☆☆☆☆
- ⚡️ **効果性:** ⭐⭐⭐⭐⭐
- 🚀 **活用度:** ⭐⭐⭐⭐⭐

> _「動くことは動くけど… なんで動いてるの？」_

開発者の永遠のミステリーです。Stack Overflowから持ってきたコードがなぜか動くものの、どこをどう直せば自分のプロジェクトに適用できるのか分からない経験はありませんか？あるいは、前任者が残していった「暗号文」のようなレガシーコードを解読しなければならない状況に陥っていませんか？

このプロンプトは、そんなあなたの**専属AIメンター**となり、難解なコードを優しく丁寧に紐解いてくれます。

---

## ⚡️ 3行まとめ (TL;DR)

1. コード全体のロジックと目的を分かりやすく要約して解説します。
2. 難解な部分ごとに**注釈（コメント）**を追加し、行単位で詳細に分析します。
3. 初心者が見落としがちな特有の文法や関数についても追加で説明します。

## 🚀 解決策：「コード難読化解除機」プロンプト

### 🥉 Basic Version (基本形)

手っ取り早く結果だけが知りたい場合に使用してください。

> **役割:** あなたは`[親切なシニアエンジニア]`です。
> **タスク:** 以下の`[コード]`がどのような処理を行っているか、一行ずつ注釈（コメント）を付けて初心者にも分かりやすく説明してください。

<br>

### 🥇 Pro Version (専門家形)

より詳細で深い理解を求める場合に使用してください。

> **役割 (Role):** あなたは`[親切なシニアエンジニアであり、私の専属メンター]`です。
>
> **状況 (Context):**
>
> - 背景: 私はこのコードが正確に何をするのか、どんな原理で動いているのかを理解したいと考えています。
> - 目標: 私は`[初級/中級]`レベルのエンジニアです。私の目線に合わせて、分かりやすく解説してください。
>
> **タスク (Task):**
>
> 1. **[要約]**: このコードのコアとなる機能と処理の流れを3つの文で要約してください。
> 2. **[行単位の分析]**: コードのすべての行（または重要なブロック）に**丁寧な注釈（コメント）**を付けて書き直してください。
> 3. **[文法ポイント]**: ここで使われている特有の文法やライブラリの関数があれば、別途詳しく説明してください（例：リスト内包表記、アロー関数、正規表現など）。
>
> **制約事項 (Constraints):**
>
> - 難解な専門用語の羅列は避け、例え話などを交えて直感的な表現を使用してください。
> - 各変数がどのような役割を担っているのかを明確に指摘してください。
>
> **入力コード:**
>
> `[ここにコードを貼り付けてください]`

## 💡 筆者のインサイト (Insight)

このプロンプトが特に威力を発揮するのは、**「他人の書いた正規表現」や「複雑なネストを持つデータ処理コード」を解読する時**です。

エンジニアとして成長する過程で、ゼロからコードを書く時間よりも、既存のコードを読んで理解する時間の方が圧倒的に多くなります。AIにただ「これ何？」と聞くのではなく、「私のレベルに合わせて」「行単位で注釈を付けて」と具体的に指示することで、単なる答え合わせではなく**「生きた学習教材」**を手に入れることができます。

実際に実務でレガシーシステムを引き継いだ際、このプロンプトを使って数千行のコードを少しずつ解読していくことで、業務の引き継ぎ時間を半分以下に短縮できました。

## 🙋 よくある質問 (FAQ)

- **Q: 無料版のChatGPTでも十分に解説してくれますか？**
  - A: はい、基本的なロジックの解説であれば無料版でも十分に役立ちます。ただし、より長く複雑なコードや、最新のフレームワークに関する解説を求める場合は、Claude 3.5 SonnetやGPT-4oなどの高性能モデルを使用することをおすすめします。

- **Q: どんなプログラミング言語でも対応可能ですか？**
  - A: Python、JavaScript、Java、C++などの主要言語はもちろん、SQLの複雑なクエリやシェルスクリプトの解説にも非常に有効です。

## 🧬 プロンプト解剖 (Why it works?)

1. **メンターとしての役割付与:** AIに「シニアエンジニア」という役割を与えることで、単なる機械的な翻訳ではなく、後輩を指導するような温かく分かりやすいトーンを引き出しています。
2. **学習者のレベルの明示:** `[初級/中級]`と指定することで、AIが前提知識をどこまで省略して良いかを判断し、難しすぎず易しすぎない絶妙な解説を提供します。
3. **構造化された出力:** 「要約」「行単位の分析」「文法ポイント」と出力を3段階に分けることで、全体像の把握から詳細な理解まで、人間の認知プロセスに沿った学習が可能になります。

## 📊 証明: Before & After

### ❌ Before (入力 - Python)

```python
result = [x * 2 for x in range(10) if x % 2 == 0]
```

### ✅ After (プロンプト適用結果)

**[要約]**

1. 0から9までの数字の中から、偶数だけを抽出します。
2. 抽出した偶数の値をそれぞれ2倍にします。
3. 最終的に計算された値を持つ新しいリストを作成するコードです。

**[行単位の分析]**

```python
# range(10)によって0から9までの数字を順番に変数xに代入します。
# if x % 2 == 0: もしxを2で割った余りが0（つまり偶数）であれば、
# x * 2: そのxを2倍して、新しいリストの要素として追加します。
result = [x * 2 for x in range(10) if x % 2 == 0]
```

**[文法ポイント]**

- **リスト内包表記 (List Comprehension):** Python特有の強力な文法です。通常であれば数行にわたって書く `for` ループと `if` 条件文を、たった1行で簡潔かつ高速に処理できる記述方法です。
- **構造:** `[式 for 変数 in イテラブルオブジェクト if 条件]`
