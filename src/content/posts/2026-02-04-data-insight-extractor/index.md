## (Updated) (Updated) 🚀 해결책: "데이터 분석가" {#updated}

### (Updated) (Updated) 🥉 Basic Version (기본형) {#updated}

빠르게 결과만 필요할 때 사용하세요.

> **역할:** 너는 데이터 사이언티스트야.
> **요청:** 숫자만 가득한 엑셀 데이터, AI에게 분석시키고 시사점 도출하기해줘.

<br>

### (Updated) (Updated) 🥇 Pro Version (전문가형) {#updated}

디테일한 퀄리티가 필요할 때 사용하세요.

아래 **PROMPT** 내용을 복사해서 사용하세요.

> **역할 (Role):** 너는 데이터 사이언티스트야.
>
> **상황 (Context):** 다음은 우리 회사의 지난달 [데이터 종류]야. (데이터 붙여넣기)
>
> **요청 (Task):**
>
> 1. 이 데이터에서 눈에 띄는 패턴이나 특이점(Trend/Outlier)을 3가지 찾아줘.
> 2. 각 패턴이 비즈니스적으로 어떤 의미가 있는지 해석해줘.
> 3. 이를 바탕으로 개선해야 할 점이나 기회 요소를 제안해줘.

- **제약사항 (Constraints):**
  > - 수치적인 근거를 반드시 포함할 것.
- 전문 용어보다는 비즈니스 언어로 풀어서 설명할 것.


### (Updated) (Updated) ✅ After (결과) {#updated}

```text

2. 재구매율이 전월 대비 5% 하락 -> 기존 고객 관리 프로그램 점검 필요
```
