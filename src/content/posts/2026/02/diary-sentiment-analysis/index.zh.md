# 📔 日记分析：导入一个月日记，用 AI 绘制你的心理变化曲线

- **🎯 推荐对象：** 每天坚持写日记却从不回看的记录者，以及想弄清“最近为什么总是职业倦怠”的职场人
- **⏱️ 所需时间：** 5分钟（文本提取与 AI 分析）
- **🤖 推荐模型：** ChatGPT Plus（需开启 GPT-4o 及高级数据分析功能）

- ⭐ **难度级别：** ⭐⭐☆☆☆
- ⚡️ **效果显著：** ⭐⭐⭐⭐⭐
- 🚀 **实用程度：** ⭐⭐⭐⭐⭐

> _“每晚写日记倾吐心声，但你是否客观审视过自己的情绪是如何崩溃、又是如何自愈的？”_

记录只有在被分析时才能成为资产，否则它们只是堆在抽屉里的沉重包袱。现在，试着将**一个月的日记数据**交给 AI 吧。利用强大的文本挖掘（Text Mining）技术，AI 能为你绘制出隐藏在文字背后的真实情绪曲线，并将你潜意识里反复使用的词汇以词云的形式可视化。是时候去破解那些连你自己都未曾察觉的心理算法了。

---

## ⚡️ 三行总结 (TL;DR)

1. 从你常用的日记 App 或 Notion 中，将过去一个月的日记导出为 `txt` 或 `csv` 格式。
2. 将文件上传至 ChatGPT，并输入**数据可视化与情感分析**提示词。
3. 查阅 AI 生成的情绪变化折线图及压力关键词报告，进行一场高效的深度复盘。

## 🚀 解决方案："Mind Tracker" 情绪追踪分析提示词

### 🥉 Basic Version (基础版)

当你只想快速了解自己潜意识中的主要关注点和情绪状态时，请使用此版本。

> **角色：** 你是一位数据分析师兼心理咨询师。
> **请求：** 请从我上传的这一周日记文本中，提取出出现频率最高的前 5 个名词和形容词。基于这些词汇，请用一句话精准总结我目前的心理状态。


### 🥇 Pro Version (专业版)

此为专业版提示词。它不仅能提供总结，还能通过科学的数据可视化，深入剖析你每日情绪的波动幅度及其根本原因。

> **角色 (Role)：** 你是一位拥有 10 年经验的数据科学家兼临床心理学家。你非常擅长通过分析用户的文本数据来发现其心理模式，并将其可视化。
> 
> **背景 (Context)：**
> 
> - 背景：我虽然每天都在写日记，但情绪起伏很大，我希望通过客观的数据来找出背后的原因。
> - 目标：基于我的日记数据，绘制出一整个月的心理变化曲线图，并深入挖掘导致压力的根本原因以及情绪恢复的模式。
> 
> **任务 (Task)：**
> 
> 1. **情绪量化：** 请分析我提供的日记数据，将我每天的情绪状态进行量化打分，范围从 -10（极度抑郁/压力爆表）到 +10（极致幸福/成就感满满）。
> 2. **数据可视化（运行 Python）：** 请使用**折线图 (Line Chart)** 将每天的情绪变化直观地展现出来。
>    - x 轴设定为“日期”，y 轴设定为“情绪分数”。
>    - 请调用 Python 的 `matplotlib` 或 `seaborn` 库，渲染出清晰易懂、美观的图表。
> 3. **深层原因剖析：** 请分析情绪分数达到最低点（谷底）和最高点（巅峰）那几天的日记内容，向我解释究竟是哪些事件或关键词对我的情绪产生了最直接、最强烈的影响。
> 4. **行为处方：** 为了让我下个月能更好地管理心态，请基于数据，为我提供 3 个我应该刻意“增加”的行为，以及 3 个我必须“避免”的行为。
> 
> **约束条件 (Constraints)：**
> 
> - 所有的分析结果必须**严格基于**用户提供的文本数据，不可凭空捏造。
> - 在生成可视化图表时，请务必设置好支持中文字体的配置（例如：`plt.rcParams['font.sans-serif'] = ['SimHei']`），以防图表中文本出现乱码（方块字）。
> - 最终交付物请使用 Markdown 格式排版，确保结构清晰、像一份专业的分析报告。
> 
> **警告 (Warning)：**
> 
> - 绝对不要对我的日记内容进行任何道德审判或价值观评判。请始终保持客观分析且富有同理心的专业态度。

## 💡 作者深度洞察 (Writer's Insight)

这个提示词真正的价值在于，它用数据将我们的**“元认知 (Meta-cognition)”**具象化了。我也曾经历过被工作压得喘不过气，每天只能在日记里倾泻“好累、想辞职”的时期。然而，当我用这套提示词分析了一个月的日记后，我惊讶地发现——那些情绪跌入谷底的日子，都有一个共同的触发点：“严重睡眠不足”叠加“与某位特定客户的会议”。

如果你没有记日记的习惯，不妨尝试导出**“文件传输助手”或“仅有自己的微信群”**的聊天记录进行分析。或者，分析一下你在工作软件（如 Slack、钉钉、飞书）中发出的消息。你会赤裸裸地看到，自己在哪个时间段最具攻击性，或者在处理什么业务时使用的积极词汇最多。人类的记忆会被情绪篡改，但数据从不说谎。

## 🙋 常见问题解答 (FAQ)

- **Q: 阿尔法狗时代的纸质笔记本，手写的日记该如何分析？**
  - A: 直接用手机拍下日记本的页面，然后将图片发送给 ChatGPT 即可。GPT-4o 强大的 OCR（光学字符识别）功能，即使是潦草的字迹也能精准提取为文本，并进行同等深度的分析。

- **Q: 把如此私密的日记交给 AI，存在隐私泄露的风险吗？**
  - A: 你的担忧非常合理。在开始分析之前，请务必前往 ChatGPT 的 设置 (Settings) > 数据控制 (Data Controls) 中，**关闭“使用我的内容改进模型 (Improve the model for everyone)”**的选项。为了确保万无一失，强烈建议在上传文件前，将文中的真实姓名或极其敏感的专有名词替换为“A”、“B”进行脱敏处理。

- **Q: AI 生成的图表里，中文字符全变成了方块（□）乱码怎么办？**
  - A: 这是 ChatGPT 运行 Python 环境绘制图表时的常见问题。只要在提示词中明确指示：“*在绘制图表时，请配置中文字体，如 `plt.rcParams['font.sans-serif'] = ['SimHei']` 或使用其他可用的中文字体，确保中文不乱码*”，就能完美解决。我们在 Pro 版提示词中已经为你预设了这一条件。

## 🧬 提示词解剖 (Why it works?)

1. **强制调用 Python 环境 (`数据可视化（运行 Python）`)：** 我们不仅仅是在让 AI 玩文字游戏，而是强制它启动内置的 Python 沙盒（Advanced Data Analysis）来生成物理层面的图像。比起密密麻麻的文字报告，一张直观的情绪折线图能给复盘带来降维打击般的震撼。
2. **定量数据降维 (`量化打分`)：** 将充满主观色彩的情绪文本，转化为 AI 能够进行数学计算的客观定量数据。这一标尺越清晰，得出的曲线图就越具公信力。
3. **闭环行动指南 (`行为处方`)：** 分析过去是为了指导未来。通过要求 AI 输出具体的 To-Do / Not-To-Do 清单，我们将单纯的“情感树洞”升华为了极具**实用效能 (Pragmatic Utility)** 的个人成长引擎。
