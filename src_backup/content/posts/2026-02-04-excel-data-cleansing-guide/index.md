데이터 분석보다 더 오래 걸리는 게 데이터 전처리(Data Cleaning)입니다. 형식이 제각각인 데이터를 통일시키는 건 엄청난 노가다입니다.

## (Updated) (Updated) (Updated) 🚀 해결책: "데이터 클리너"

### (Updated) (Updated) (Updated) 🥉 Basic Version (기본형)

빠르게 결과만 필요할 때 사용하세요.

> **역할:** 너는 엑셀 데이터 처리 전문가야.
> **요청:** 주소 분리, 날짜 형식 통일 등 데이터 클렌징 방법 찾기해줘.

<br>

### (Updated) (Updated) (Updated) 🥇 Pro Version (전문가형)

디테일한 퀄리티가 필요할 때 사용하세요.

아래 **PROMPT** 내용을 복사해서 사용하세요.

> **역할 (Role):** 너는 엑셀 데이터 처리 전문가야.
>
> **상황 (Context):** 엑셀 A열에 [데이터 예시]처럼 데이터가 섞여 있어. 이걸 B열(주소), C열(우편번호)로 나누고 싶어.
>
> **요청 (Task):**
>
> 1. 텍스트 나누기나 함수(LEFT, RIGHT, MID, FIND 등)를 사용해 데이터를 분리하는 방법을 알려줘.
> 2. 빠른 채우기(Flash Fill) 기능을 활용할 수 있다면 단축키와 방법을 알려줘.
> 3. 혹시 데이터에 예외가 있을 경우를 대비한 검증 방법을 알려줘.

- **제약사항 (Constraints):**
  > - 초보자도 이해하기 쉽게 단계별로 설명할 것.
- 가장 효율적인 방법 1가지만 추천할 것.

## (Updated) (Updated) (Updated) 🧬 프롬프트 해부 (Why it works?) {#why-it-works}

이 프롬프트는 명확한 페르소나 설정과 구체적인 지시사항(Task)을 조합하여 AI의 의도를 정확히 파악하도록 설계되었습니다. 단계별 사고(Chain of Thought)를 유도하여 논리적인 답변을 얻을 수 있습니다.

## (Updated) (Updated) (Updated) 📊 증명: Before & After

단순한 명령어와 최적화된 프롬프트의 차이는 명확합니다. 제공된 프롬프트를 사용했을 때 더 구체적이고 구조화된 답변을 얻을 수 있으며, 재수정(Re-prompting)에 드는 시간을 절약할 수 있습니다.

### (Updated) (Updated) (Updated) ❌ Before (입력)

```text
"서울시 강남구 역삼동 (12345)" 이런 식인데 괄호 안 숫자만 빼고 싶어.
```

### (Updated) (Updated) (Updated) ✅ After (결과)

```text
가장 쉬운 방법은 "빠른 채우기(Ctrl+E)"입니다.

2. B2 셀을 클릭하고 Ctrl+E를 누르세요. 끝!
```
