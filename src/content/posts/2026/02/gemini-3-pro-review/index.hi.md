## ЁЯУК рдкреНрд░рдорд╛рдг: Before & After (рдкрд╛рдпрдерди рдСрдкреНрдЯрд┐рдорд╛рдЗрдЬрд╝реЗрд╢рди)

### тЭМ Before (рдЗрдирдкреБрдЯ - рдзреАрдорд╛ $O(n^2)$ рд▓реВрдк рдХреЛрдб)

```python
import pandas as pd
import numpy as np
import time

# рдбрдореА рдбреЗрдЯрд╛
df = pd.DataFrame({'A': np.random.rand(10000), 'B': np.random.rand(10000)})

start = time.time()
# рдЕрдХреНрд╖рдо рд▓реВрдкрд┐рдВрдЧ (Inefficient looping)
results = []
for index, row in df.iterrows():
    if row['A'] > 0.5:
        results.append(row['A'] * row['B'])
    else:
        results.append(row['A'] + row['B'])
df['Result'] = results
print(f"Time taken: {time.time() - start:.2f} seconds")
# Time taken: ~4.20 seconds
```

### тЬЕ After (рдЬреЗрдорд┐рдиреА 3 рдкреНрд░реЛ рдХрд╛ рдкрд░рд┐рдгрд╛рдо - рд╡реИрдХреНрдЯрд░рд╛рдЗрдЬреЗрд╢рди)

```python
import pandas as pd
import numpy as np
import time

df = pd.DataFrame({'A': np.random.rand(10000), 'B': np.random.rand(10000)})

start = time.time()
# рдЬреЗрдорд┐рдиреА 3 рдкреНрд░реЛ рджреНрд╡рд╛рд░рд╛ рд╡реИрдХреНрдЯрд░рд╛рдЗрдЬреНрдб рд╕рдорд╛рдзрд╛рди (Vectorized solution)
df['Result'] = np.where(df['A'] > 0.5, df['A'] * df['B'], df['A'] + df['B'])
print(f"Time taken: {time.time() - start:.4f} seconds")
# Time taken: ~0.08 seconds (52x Faster)
```
