```css
  .card.is-active .card-title {
    color: #ff0055;
  }
}
/* 背后还需要配合大量的 JavaScript 逻辑来频繁切换 'is-active' 类名 */
```

### ✅ After (Interop 2026 现代标准)

```css

  container-type: inline-size;
}

@container (max-width: 600px) {
  .card:state(active) .card-title {
    color: #ff0055;
  }
}

.card:state(active) .card-title {
  color: #00f3ff;
}
```
