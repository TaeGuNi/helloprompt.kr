## 🧬 提示词解剖 (Why it works?)

---

## 📊 证明: Before & After

### ❌ Before (输入)

```text
帮我写个韩国手机号正则。连字符有没有都行。除了010，011也可以。
```

<br>

### ✅ After (结果)

```javascript
// JavaScript 示例

const regex = /^01([0|1|6|7|8|9])-?([0-9]{3,4})-?([0-9]{4})$/;

// 测试
const phone1 = "010-1234-5678"; // 匹配 O
const phone2 = "01012345678"; // 匹配 O
const phone3 = "02-123-4567"; // 匹配 X

console.log(regex.test(phone1)); // true
```

### 🔍 解说

- `^`: 字符串的开始
- `01([0|1|6|7|8|9])`: 以01开头，后面跟着0,1,6,7,8,9之一 (识别号)
- `-?`: 连字符(-)有0个或1个 (可选)
- `([0-9]{3,4})`: 数字0-9有3个或4个 (中间位)
- `([0-9]{4})`: 数字0-9有4个 (末位)
- `$`: 字符串的结束
