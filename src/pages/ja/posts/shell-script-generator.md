## 📊 証明: Before & After

### ❌ Before (入力要求事項)

「現在フォルダにあるすべてのJPGファイルをimagesフォルダへ移して、ファイル名を日付\_元の名前.jpgに変えて。」

### ✅ After (結果)

```bash
#!/bin/bash
set -e  # エラー発生時スクリプト中断

TARGET_DIR="images"
DATE_PREFIX=$(date +%Y%m%d)

# 1. 対象ディレクトリがなければ生成
if [ ! -d "$TARGET_DIR" ]; then
    echo "📁 '$TARGET_DIR' ディレクトリを生成します..."
    mkdir -p "$TARGET_DIR"
fi

# 2. JPG ファイル移動および名前変更
echo "🚀 ファイル移動を開始します..."
for file in *.jpg; do
    if [ -f "$file" ]; then
        new_name="${DATE_PREFIX}_${file}"
        mv "$file" "$TARGET_DIR/$new_name"
        echo "✅ Moved: $file -> $TARGET_DIR/$new_name"
    fi
done

echo "🎉 すべての作業が完了しました！"
```
